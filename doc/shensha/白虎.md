# 白虎

## 定義

白虎為四靈之一（青龍、白虎、朱雀、玄武），在命理神煞中白虎主凶，代表刀兵、血光、意外傷災之象。凡命中帶白虎，須特別注意人身安全，防範意外血光之災。

## 性質

**類型**：凶星

**主要含義**：主血光意外、須防刀傷車禍、宜謹慎行事。白虎主兇煞，與血光相關，凡逢此星，易有外傷、手術、車禍等血光之事。

## 起例方法

以**年支**查對應地支，凡四柱（月、日、時）地支中出現對應地支，即為命中帶白虎。

| 年支 | 白虎地支 |
|-----|---------|
| 子  | 申      |
| 丑  | 酉      |
| 寅  | 戌      |
| 卯  | 亥      |
| 辰  | 子      |
| 巳  | 丑      |
| 午  | 寅      |
| 未  | 卯      |
| 申  | 辰      |
| 酉  | 巳      |
| 戌  | 午      |
| 亥  | 未      |

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 幼年易有血光意外，或祖上有刀傷之事 |
| 月柱   | 青壯年須防工傷、車禍、手術 |
| 日柱   | 本人一生易逢血光，配偶亦需注意安全 |
| 時柱   | 晚年須防意外，子女也需注意安全事項 |

## 傳統解說

白虎在傳統五行方位中屬西方金星，金主肅殺，故白虎主刀兵血光之災。命理古籍云：「白虎當頭坐，無災必有禍。」命中逢白虎，易有外科手術、意外受傷、車禍血光等事。然若命局中有制化白虎之神（如火克金），則凶性可減。從事醫療、軍警、屠宰等行業者，白虎反可化凶為吉，視為職業特質。
