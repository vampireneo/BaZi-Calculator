# 浮沉

## 定義

浮沉為八字神煞之一，屬**凶煞**。浮沉如水中浮木，主起伏不定。命帶此煞者，事業與生活難以穩定，浮沉不定，難有長久的安定局面。

## 含義

- 主浮沉不定、事業起伏、難有穩定
- 人生如浪潮起伏，高低落差明顯
- 居無定所，難以在一地長久安定

## 起例方式

依**年支**查對應地支，若該地支出現在命盤四柱中，即為帶浮沉。

### 年支與浮沉對照表

| 年支 | 浮沉地支 |
|------|---------|
| 子   | 巳 |
| 丑   | 午 |
| 寅   | 未 |
| 卯   | 申 |
| 辰   | 酉 |
| 巳   | 戌 |
| 午   | 亥 |
| 未   | 子 |
| 申   | 丑 |
| 酉   | 寅 |
| 戌   | 卯 |
| 亥   | 辰 |

### 計算步驟

1. 取命盤年柱地支（年支）
2. 查上表得浮沉對應地支
3. 若月柱、日柱或時柱地支中有對應地支，即帶浮沉

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 丙   | **寅**（查表：浮沉為未）|
| 月柱 | 戊   | 午   |
| 日柱 | 庚   | **未**（命中浮沉）|
| 時柱 | 壬   | 申   |

年支「寅」對應浮沉地支「未」，日柱地支為「未」，故帶浮沉。

## 各柱浮沉效應

| 浮沉所在柱位 | 影響範疇 |
|-------------|---------|
| 年柱 | 幼年動盪，家庭環境不穩 |
| 月柱 | 事業多起伏，職業易變動 |
| 日柱 | 婚姻感情不穩，易分離聚合 |
| 時柱 | 晚年不安，居所多遷移 |

## 傳統解說

浮沉之象，猶如一塊浮木在水面隨波逐流，時而漂至高處，時而沉落低谷，難以自主掌控方向。命帶浮沉者，宜培養內在定力，建立穩固的生活基礎，學會在動盪中保持平靜，方能在起伏之中尋得立足之地。
