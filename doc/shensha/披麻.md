# 披麻

## 定義

披麻，又稱「披麻戴孝」之象，為主喪憂孝服之凶煞。「披麻」取自古代喪禮中披麻布為孝服之習俗，凡命中帶此煞，主易逢喪事孝服、憂患纏身。

## 性質

**類型**：凶星

**主要含義**：主麻煩纏身、須防孝服、憂患多見。披麻戴孝之象，主喪憂之事，凡逢此煞，易有親人離世或喪葬相關之憂患。

## 起例方法

以**年支**查對應地支，凡四柱（月、日、時）地支中出現對應地支，即為命中帶披麻。

| 年支 | 披麻地支 |
|-----|---------|
| 子  | 亥      |
| 丑  | 子      |
| 寅  | 丑      |
| 卯  | 寅      |
| 辰  | 卯      |
| 巳  | 辰      |
| 午  | 巳      |
| 未  | 午      |
| 申  | 未      |
| 酉  | 申      |
| 戌  | 酉      |
| 亥  | 戌      |

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 祖先或幼年家中有喪事，或帶孝出生 |
| 月柱   | 父母輩或中年易逢喪服之事 |
| 日柱   | 本人一生中易多逢喪事，情緒易受憂悶 |
| 時柱   | 晚年或子女輩易有孝服纏身之事 |

## 傳統解說

披麻源自古代喪禮制度，服喪者身著麻布孝衣，故以「披麻」名此凶煞。傳統命理認為，命中帶披麻者，一生中與喪葬之事較為緣深，易為親友辦理後事，或常需奔赴喪禮。此煞與喪門同類，若兩者同時出現，則喪憂之事更為顯著。逢披麻之年，宜少參與婚慶喜事，多加保重身心。
