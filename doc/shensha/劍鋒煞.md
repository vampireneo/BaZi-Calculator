# 劍鋒煞

## 定義

劍鋒煞為八字神煞之一，屬**凶煞**。「劍鋒」意指刀劍之鋒芒，主命主易遭刀劍之災，或與外科手術、意外傷害相關，須格外謹慎。

## 含義

- 主刀劍之災、意外傷害
- 易有開刀手術、外科治療之經歷
- 從事有刀具風險職業者須倍加小心
- 宜注意交通安全，避免利器傷害

## 起算方式

以**日干**查劍鋒煞所在之地支，若該地支出現於命盤其他柱位，即為劍鋒煞。

## 各柱位影響

| 柱位 | 影響說明 |
|------|----------|
| 年柱 | 幼年易有刀傷意外，家族有此患 |
| 月柱 | 事業宜避刀具風險，中年防外傷 |
| 日柱 | 本人須防刀傷手術，配偶同論 |
| 時柱 | 晚年注意外傷，子女亦須留意 |

## 傳統解說

劍鋒煞古謂刀光劍影之煞，與金屬利器密切相關。命帶此煞者，一生中與刀劍、金屬器具之緣較深，輕則有小傷，重則可能面臨重大外科手術。然而，若命主從事外科醫師、廚師、理髮師、雕刻師等需使用刀具的行業，此煞有時亦可轉化為職業上的助力，化凶為吉。宜避免不必要的危險動作，謹慎處理利器，以減低傷害風險。
