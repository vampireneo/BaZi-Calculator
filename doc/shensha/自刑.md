# 自刑

## 定義

自刑（Zì Xíng）為八字地支神煞之一，屬**凶煞**。自刑乃地支自我刑傷之象，主自我克制過度、心性不定、易走極端，命帶自刑者易鑽牛角尖，傷人傷己。

## 含義

- 主自我刑傷、心性不定
- 易鑽牛角尖，思想走極端
- 自我要求過高，易造成心理壓力
- 嚴重時有自我傷害或自毀前程之象

## 起法（以四柱地支查）

自刑以**特定地支重複出現**於四柱中論之。能產生自刑的地支共四種：

| 自刑地支 | 五行屬性 | 說明 |
|---------|---------|------|
| 辰 | 土（水庫）| 辰辰相刑，主自我封閉 |
| 午 | 火 | 午午相刑，主性情急躁過激 |
| 酉 | 金 | 酉酉相刑，主自我苛責、潔癖 |
| 亥 | 水 | 亥亥相刑，主思慮過度、自我困擾 |

**查法**：上述四種地支（辰、午、酉、亥）若在四柱（年支、月支、日支、時支）中出現**兩次或以上**，即構成自刑。

## 各支自刑特性

| 地支 | 自刑特性 |
|------|---------|
| 辰辰 | 自我封閉，固執難化，守舊保守 |
| 午午 | 情緒波動大，急躁易怒，衝動行事 |
| 酉酉 | 完美主義，自我苛責，難以滿足 |
| 亥亥 | 思慮過度，憂鬱傾向，自我困擾 |

## 傳統解說

自刑為地支中最獨特的刑煞，非外力所傷，而是自我加諸之痛。古籍云：「自刑入命，自損自傷，非他人之故，乃自取其禍。」命帶自刑者最大的敵人往往是自己，易在思想上走入死角，在行動上做出極端之舉。此煞需命局有疏導之力（如食傷通關、印星化解），方能引導自刑之能量走向正面，化為自律、自省之力。
