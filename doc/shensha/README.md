## 計算原則

1. **以日干為主**：天乙貴人、文昌貴人、祿神、羊刃等均以日干查詢
2. **以年支為主**：三合系（桃花、驛馬、將星等）均以年支查詢
3. **以月支為主**：天德貴人、月德貴人、天醫等均以月支查詢
4. **特定柱位**：截空、月破、歲破等需排除特定柱位
5. **干支組合**：魁罡、六秀日、八專日等需干支完整組合配對

## 吉凶判斷原則

- 吉神多者，行運吉時效果倍增
- 凶煞多者，須注意行運逢沖剋時的風險
- 吉凶神同宮，吉中帶凶，需視命局整體而定
- 凶煞有制化（如生剋），其凶力可減輕
