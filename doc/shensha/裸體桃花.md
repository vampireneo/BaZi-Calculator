# 裸體桃花

## 定義

裸體桃花是當日干所對應的沐浴位，出現於**日柱**或**時柱**的地支時，所形成的特殊桃花神煞。「裸體」形容沐浴之氣外露無遮，桃花之力顯現於外，主桃花公開外露之象。

**類型**：中
**主要含義**：主桃花外露、易招桃色是非。裸體桃花指沐浴之氣外露，桃花顯於外，感情吸引力強但易惹麻煩。

---

## 計算方法

依日干確認沐浴位，若**日支**或**時支**等於該沐浴位，即為裸體桃花：

| 日干 | 沐浴位 | 日支或時支條件 |
|------|--------|----------------|
| 甲 | 子 | 日支＝子 或 時支＝子 |
| 乙 | 巳 | 日支＝巳 或 時支＝巳 |
| 丙 | 卯 | 日支＝卯 或 時支＝卯 |
| 丁 | 申 | 日支＝申 或 時支＝申 |
| 戊 | 卯 | 日支＝卯 或 時支＝卯 |
| 己 | 申 | 日支＝申 或 時支＝申 |
| 庚 | 午 | 日支＝午 或 時支＝午 |
| 辛 | 亥 | 日支＝亥 或 時支＝亥 |
| 壬 | 酉 | 日支＝酉 或 時支＝酉 |
| 癸 | 寅 | 日支＝寅 或 時支＝寅 |

---

## 各位置影響

| 沐浴位置 | 影響說明 |
|----------|----------|
| 日支 | 桃花顯於配偶宮，配偶或本身桃花外露 |
| 時支 | 桃花顯於子女宮，晚年仍有異性吸引力 |
| 帶刑沖 | 桃花引發的是非更為嚴重，宜謹慎 |
| 帶合 | 桃花收斂，但仍有感情複雜之象 |

---

## 傳統解說

沐浴古稱「敗地」，有裸裎無遮之意，主人外露、風流。裸體桃花因沐浴落於日支或時支，這兩個柱位最能直接呈現個人特質與言行，故桃花之氣全然外顯，往往使人感覺魅力四射卻難以捉摸。古書警示：「裸體桃花，招蜂引蝶，是非不斷。」命帶此神煞者宜注重自身形象，在感情上保持界線，避免因過於外露的魅力而招致桃色糾紛或名譽損傷。
