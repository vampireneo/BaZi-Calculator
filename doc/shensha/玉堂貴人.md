# 玉堂貴人

## 定義

玉堂貴人，為吉神之一，以日干為基準起算。玉堂者，天上神仙居所，象徵尊貴祥瑞之地。命帶此神，主福祿雙全，貴人扶持，一生榮華。

**類型**：吉神

## 含義

- 主福祿雙全、名利雙收
- 有貴人提攜、貴人緣極佳
- 才智超群、聲譽卓著
- 官場、職場易得長官賞識，仕途順遂

## 起例方法

以**日干**查對應玉堂貴人地支，若該地支出現於四柱之中，即為玉堂貴人。

| 日干 | 玉堂貴人位 |
|------|----------|
| 甲   | 辰        |
| 乙   | 卯        |
| 丙   | 寅        |
| 丁   | 亥        |
| 戊   | 寅        |
| 己   | 亥        |
| 庚   | 申        |
| 辛   | 酉        |
| 壬   | 戌        |
| 癸   | 未        |

**例**：日干為甲，玉堂貴人位為辰。若年、月、時柱地支有辰，即為玉堂貴人。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 年柱 | 祖上有蔭，出身良好，幼年逢貴，家學淵源 |
| 月柱 | 職場貴人多，事業順遂，上司器重，升遷有望 |
| 日柱 | 本命貴氣十足，配偶賢德，婚姻美滿，福祿自來 |
| 時柱 | 晚年享福，子女成才，後代顯貴，晚景榮耀 |

## 傳統解說

《神峰通考》云：「玉堂貴人，乃天上瑞星，所到之宮，福德自至，貴人自來。」玉堂貴人為諸貴人星中較為尊貴者，象徵天庭玉殿之瑞氣降臨。命帶此神者，一生多逢貴人相助，關鍵時刻往往有人出手扶持，化解困厄。其人多具高雅氣質，舉止端莊，易獲上位者青睞。若再配合天乙貴人、天德貴人等吉神同現，則貴氣更重，一生榮華富貴，名利雙收，為命中難得之吉象。
