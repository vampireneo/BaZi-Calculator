# 月破

## 定義

月破，是指與月柱地支相「六沖」的地支出現於日柱或時柱，形成月支被沖破之格局。六沖者，地支對沖，力量衝突，主破敗之象。

**類型**：凶神

## 含義

- 主諸事不順、計畫易受阻
- 易有破敗損失、財物耗散
- 不宜進取冒險、宜守不宜攻
- 人際關係易起衝突，合作事項易告吹

## 起例方法

以**月柱地支**查其六沖之地支，若日柱或時柱地支與月支相沖，即為月破。

| 月支 | 月破位（六沖） |
|------|-------------|
| 子   | 午           |
| 丑   | 未           |
| 寅   | 申           |
| 卯   | 酉           |
| 辰   | 戌           |
| 巳   | 亥           |
| 午   | 子           |
| 未   | 丑           |
| 申   | 寅           |
| 酉   | 卯           |
| 戌   | 辰           |
| 亥   | 巳           |

**例**：月支為寅，月破位為申。若日柱或時柱地支為申，即構成月破。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 日柱 | 本人易逢破敗，婚姻感情受衝，事業難穩固 |
| 時柱 | 晚年易有耗損，子女宮受衝，晚景宜謹慎 |

> 月破主要以日、時兩柱地支是否與月支相沖為判斷依據。

## 傳統解說

《神峰通考》云：「月破之地，萬事難成，逢之如石投水，了無痕跡。」月支為一月之主氣，代表當令之力量。日時地支與之相沖，即形成衝破月令之勢，猶如根基被撼動，凡事皆難以為繼。命帶月破者，凡事宜三思而後行，不宜輕易做出重大決定，尤忌貿然投資、開展新業。逢月破流年，宜低調行事，保守應對，靜待時機好轉，方能化解破敗之象。
