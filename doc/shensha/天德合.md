# 天德合

## 定義

天德合為八字神煞之一，屬**吉神**，為天德貴人之六合神。主逢凶化吉、貴人相助，能化解凶煞災厄，為天德之合化神。

## 含義

- 逢凶化吉，化解災厄
- 貴人相助，遇難有救
- 品德高尚，受天地護佑
- 官司訴訟易獲有利結果
- 一生多貴人提攜

## 起法（以月支查）

天德合依**月支**起算，先找出該月之天德貴人（限天干），再取其六合天干：

| 月支 | 天德貴人 | 六合關係   | 天德合 |
|------|----------|------------|--------|
| 寅月 | 丁       | 丁壬合     | 壬     |
| 辰月 | 壬       | 壬丁合     | 丁     |
| 巳月 | 辛       | 辛丙合     | 丙     |
| 未月 | 甲       | 甲己合     | 己     |
| 申月 | 癸       | 癸戊合     | 戊     |
| 戌月 | 丙       | 丙辛合     | 辛     |
| 亥月 | 乙       | 乙庚合     | 庚     |
| 丑月 | 庚       | 庚乙合     | 乙     |

> **注意**：卯月、午月、酉月、子月之天德貴人為地支，無對應天干，故此四月無天德合。

## 各柱影響

| 所在柱位 | 影響 |
|----------|------|
| 年柱     | 祖德深厚，家族受天地護佑 |
| 月柱     | 事業有貴人提攜，職涯順遂 |
| 日柱     | 本人受天德護佑，婚姻有貴人助力 |
| 時柱     | 子女受庇佑，晚年逢難有救 |

## 與天德貴人之關係

天德合為**天德貴人**之六合化神：

- **天德貴人**：直接承天德之力，化煞力量最強
- **天德合**：以合化之力發揮，力量略遜但同樣吉祥
- 兩者同現，化凶解厄之力最為完整

## 傳統解說

天德為天之德神，主掌天地間的善德之氣。其合化神天德合同樣承載上天護佑之力，古籍稱「天德合，百事無礙」。命帶天德合者，縱使命局中有凶煞沖剋，亦能逢難呈祥、逢凶化吉。此吉神尤利於官場仕途、法律訴訟，以及化解飛天煞、羊刃等凶煞之力。
