# 文昌貴人

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 吉神 |
| 別名 | 文昌星 |
| 取用基準 | 日干 |

## 定義與含義

文昌貴人為命盤中象徵聰明智慧與學識才華的吉神。命帶文昌者，主聰明智慧、學業有成、利於考試功名，思維敏銳，善於學習，適合從事學術、教育、文化、寫作等相關領域。

## 起算方法

以**日干**為基準，查看四柱地支中是否出現對應地支。

| 日干 | 文昌貴人地支 |
|------|------------|
| 甲 | 巳 |
| 乙 | 午 |
| 丙 | 申 |
| 丁 | 酉 |
| 戊 | 申 |
| 己 | 酉 |
| 庚 | 亥 |
| 辛 | 子 |
| 壬 | 寅 |
| 癸 | 卯 |

## 判斷步驟

1. 找出日干（日柱天干）
2. 對照上表查出對應地支
3. 在年、月、日、時四柱的地支中搜尋該地支
4. 出現即代表命帶文昌貴人

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 祖先積德，家學淵源，幼年聰穎 |
| 月柱 | 求學運佳，學業順遂，中年得益 |
| 日柱 | 個人智慧過人，本身才華出眾 |
| 時柱 | 晚年有文名，子女聰明，老來受人敬重 |

## 傳統解說

文昌為文曲星之化身，古人視之為主掌文運、科舉的神煞。命帶文昌貴人者，天生具備讀書學習的天賦，記憶力強，理解能力佳，口才好，善於表達。

在現代命理應用上，文昌貴人利於一切需要智識、考試與文書的事項，如升學考試、職業資格考、著作出版等。若文昌貴人同時與天乙貴人或天德貴人同柱，吉力更強。

> **注意**：文昌貴人以不受沖破為佳，若文昌所在之地支被沖剋，則吉力受損，學業考運較不穩定。
