# 六害

## 定義

六害（Liù Hài）為八字地支神煞之一，屬**凶煞**，又稱「六穿」。六害主阻礙與傷害，命帶六害者主六親不和、易有害人之事、須防小人暗算。

## 含義

- 主六親不和，家人、朋友間易生嫌隙
- 易遭小人陷害或暗中阻礙
- 事業上多有阻力，合作關係難以長久
- 感情上易有第三者介入或伴侶間互相傷害

## 起法（以四柱地支查）

六害以四柱地支中出現以下**配對組合**論之，任意兩柱之間有害關係即構成六害：

| 害關係 | 五行說明 |
|--------|---------|
| 子 害 未 | 子水剋未中丁火，未土又困子水 |
| 丑 害 午 | 丑土剋午中丁火，形成相互牽制 |
| 寅 害 巳 | 寅巳相刑兼相害，木火相剋牽引 |
| 卯 害 辰 | 卯木剋辰土，辰又為卯之墓庫 |
| 申 害 亥 | 申金剋亥水，水反侵金，相互牽制 |
| 酉 害 戌 | 酉金剋戌中丁火，又受戌土洩金 |

**查法**：逐一檢查四柱地支（年、月、日、時），凡有以上任一對害關係出現，即命帶六害。

## 各柱影響

| 受害柱位 | 影響方向 |
|---------|---------|
| 年月害 | 祖輩、父母緣薄，早年環境受阻 |
| 月日害 | 父母或配偶之間有衝突，中年事業受阻 |
| 日時害 | 配偶與子女緣薄，晚年家庭不和 |

## 傳統解說

六害又名六穿，穿即穿透傷害之意。古籍云：「六害相逢，六親難靠，暗中受損，防人防己。」六害的傷害往往較為隱蔽，非如六沖般直接明顯，而是如蠹蟲蛀木，緩慢而深入地侵蝕六親緣分與事業根基。命帶六害者宜廣結善緣，謹慎選擇合作對象，防範小人在背後作梗。
