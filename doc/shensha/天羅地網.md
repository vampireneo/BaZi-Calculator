# 天羅地網

## 定義

天羅地網是一組成對出現的神煞。辰為天羅，戌為地網，二者須同時出現於四柱之中方成此煞。

## 類型

**凶煞**

## 含義

主易遇困阻、諸事不順、須防官非訴訟。辰為天羅，戌為地網，二者同現則困阻更重，如同落入羅網之中，難以脫身。

## 起例方法

| 條件 | 說明 |
|------|------|
| 天羅 | 四柱地支含「辰」 |
| 地網 | 四柱地支含「戌」 |
| 成立條件 | 辰與戌**同時**出現於四柱中 |

> 注意：辰、戌必須同時出現方成天羅地網，缺一不計。

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 祖業難守，早年多阻礙 |
| 月柱 | 事業受困，職場多紛爭 |
| 日柱 | 婚姻不順，夫妻易有糾紛 |
| 時柱 | 晚年操勞，子女緣薄 |

## 傳統解釋

古籍云：「辰為天羅，戌為地網」，天羅為天之牢籠，地網為地之束縛，二者同現象徵人生處處受縛，行事多阻。命中帶此煞者，宜謹言慎行，避免涉及訴訟糾紛，凡事多留退路。修行之人或出家者，此煞反而不忌，因已超脫世俗之網。
