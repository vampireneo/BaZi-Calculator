# 弔客

## 定義

弔客，為凶神之一，與「喪門」相對，主喪服孝事。弔客取象弔喪之客，代表與喪亡、孝服相關的事件，提示命主須留意親友之凶訊。

**類型**：凶神

## 含義

- 主喪服之事、哀傷憂愁
- 須防親友有難、意外亡故
- 宜謹慎行事、避免參與喜慶
- 與喪門相關，主孝服之事

## 起例方法

以**年支**查對應弔客地支，若該地支出現於其他柱位，即為弔客。

| 年支 | 弔客位 |
|------|------|
| 子   | 戌   |
| 丑   | 亥   |
| 寅   | 子   |
| 卯   | 丑   |
| 辰   | 寅   |
| 巳   | 卯   |
| 午   | 辰   |
| 未   | 巳   |
| 申   | 午   |
| 酉   | 未   |
| 戌   | 申   |
| 亥   | 酉   |

**例**：年支為子，弔客位為戌，若月、日、時柱地支有戌，即為弔客。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 年柱 | 幼年或祖輩有喪事，家族有孝服之象 |
| 月柱 | 父母或兄弟有憂患，中年有喪服之事 |
| 日柱 | 本身或配偶有孝服之應，夫妻宮見凶 |
| 時柱 | 晚年或子女有憂患，子嗣宮見悲情 |

## 傳統解說

《神煞起例》云：「弔客主孝服，逢之親喪哭泣頻。」弔客與喪門同為主喪之神，弔客偏重外來弔唁之象，即身邊親友發生喪亡，命主需前往弔祭。命帶弔客者，一生中易遭遇親友離世之事，情感豐富，易感傷。逢弔客流年，宜避免參加喜慶婚宴，謹言慎行，保持低調，以化解凶象。
