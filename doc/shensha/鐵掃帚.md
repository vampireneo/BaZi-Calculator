# 鐵掃帚

## 定義

鐵掃帚為八字神煞之一，屬**凶煞**。鐵掃帚掃盡財祿，主破財之象，命帶此煞者主破財敗業、錢財難聚，宜節儉自守。

## 含義

- 主破財敗業、錢財難聚、宜節儉
- 財來財去，難以積累財富
- 宜量入為出，避免投機冒險

## 起例方式

依**月支**查對應地支，若該地支出現在命盤四柱中，即為帶鐵掃帚。**男命（男命）與女命（女命）使用不同對照表**，此處列出男命對照表。

### 男命（男命）對照表

| 月支 | 鐵掃帚地支 |
|------|-----------|
| 子   | 申 |
| 丑   | 酉 |
| 寅   | 戌 |
| 卯   | 亥 |
| 辰   | 子 |
| 巳   | 丑 |
| 午   | 寅 |
| 未   | 卯 |
| 申   | 辰 |
| 酉   | 巳 |
| 戌   | 午 |
| 亥   | 未 |

> **注意**：女命（女命）使用不同起例表，請依女命規則另行查對。

### 範例（男命）

某男命月支為「午」，查表得鐵掃帚對應地支為「寅」。若其四柱（年、月、日、時）中任一柱地支為「寅」，即命帶鐵掃帚。

## 各柱鐵掃帚效應

| 鐵掃帚所在柱位 | 影響範疇 |
|---------------|---------|
| 年柱 | 祖業難守，家族財運不佳 |
| 月柱 | 事業財運起伏，難以積累 |
| 日柱 | 個人財務破耗，配偶財運亦差 |
| 時柱 | 晚年財運欠佳，子女財運不助 |

## 傳統解說

鐵掃帚之象，猶如持鐵製掃帚將財祿悉數掃除，分毫不留。命帶此煞者，財來財去，即使努力賺取，仍難守財。宜培養節儉習慣，避免奢靡揮霍，並遠離投機賭博之事，方能減輕破財之患。
