# 隔角

## 定義

隔角，又稱「隔角煞」，是以年支為基準，取其前後各隔兩位之地支（即相距兩格之地支）。此兩支出現於非年柱之其他柱位，即為隔角。隔角取象被阻隔、有距離之感，主人際緣薄、隔閡難通。

**類型**：凶神

## 含義

- 主六親緣薄、與親人有隔閡
- 易有人際關係之障礙
- 溝通不暢、彼此誤解
- 人際關係需特別注意，防小人阻隔

## 起例方法

以**年支**查其前後各隔兩位之地支，若該地支出現於月、日、時柱，即為隔角。

| 年支 | 隔角位（±2位） |
|------|-------------|
| 子   | 寅、戌       |
| 丑   | 卯、亥       |
| 寅   | 辰、子       |
| 卯   | 巳、丑       |
| 辰   | 午、寅       |
| 巳   | 未、卯       |
| 午   | 申、辰       |
| 未   | 酉、巳       |
| 申   | 戌、午       |
| 酉   | 亥、未       |
| 戌   | 子、申       |
| 亥   | 丑、酉       |

**例**：年支為子，隔角位為寅與戌。若月、日、時柱地支有寅或戌，即為隔角。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 月柱 | 與父母兄弟有隔閡，職場易受阻，貴人難近 |
| 日柱 | 夫妻感情有距離，配偶緣薄，婚姻需磨合 |
| 時柱 | 親子關係疏離，子女易叛逆，晚年孤獨感重 |

## 傳統解說

《命理探源》云：「隔角主緣薄，雖同處一室，心有千里之遠。」隔角之象，如兩地之間橫亙高山，雖有心往來，卻難以暢通。命帶隔角者，往往在人際關係上有天然的隔閡感，易被誤解，亦易誤解他人。此煞在家庭關係中尤為明顯，親子、夫妻之間需要更多溝通與包容。宜主動表達心意，打破隔閡，廣結善緣，方能化解此煞之負面影響。
