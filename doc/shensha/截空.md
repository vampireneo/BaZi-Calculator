# 截空

## 定義

截空，又稱「旬空」，即六十甲子中每一旬之「空亡」地支。每旬由十個天干配十二地支，必有兩個地支無天干相配，此二支即為空亡之地，稱為截空。

**類型**：凶神（兼具玄學助益之特殊性）

## 含義

- 主空虛不實、事多阻滯、難以圓滿
- 計畫易落空、承諾難兌現
- 但亦主超脫世俗、利於宗教玄學修行
- 空亡之地，諸事落空，然宗教玄學反得助益

## 起例方法

以**日柱天干地支**所在旬首，查出對應空亡地支。若年、月、日、時柱之地支落入空亡，即為截空。

| 旬首（日柱所在旬） | 空亡地支 |
|------------------|---------|
| 甲子旬（甲子～癸酉） | 戌、亥   |
| 甲戌旬（甲戌～癸未） | 申、酉   |
| 甲申旬（甲申～癸巳） | 午、未   |
| 甲午旬（甲午～癸卯） | 辰、巳   |
| 甲辰旬（甲辰～癸丑） | 寅、卯   |
| 甲寅旬（甲寅～癸亥） | 子、丑   |

**例**：日柱為庚午，屬甲午旬，空亡為辰、巳。若年月時有辰或巳，即為截空。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 年柱 | 祖業難承、早年根基薄弱 |
| 月柱 | 事業多阻、父母緣薄 |
| 日柱 | 婚姻感情易落空、自身運勢起伏 |
| 時柱 | 子嗣緣薄、晚年規劃難圓 |

## 傳統解說

《滴天髓》云：「空亡之地，有名無實。」截空所到之處，萬事易成泡影，然「空」亦有空靈清透之義。命帶截空者，若能轉而從事宗教、玄學、哲學、藝術等超脫世俗之行業，反能借空亡之力，出塵入聖，成就非凡。世俗功名難求，精神境界卻易昇華，此截空之辯證妙用也。
