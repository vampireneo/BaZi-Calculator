# 反吟

## 定義

反吟為八字神煞之一，屬**凶煞**。「反」者，背反也；「吟」者，長嘆也。反吟如人仰天長嘆，主動盪不安，主變動不安、事多反覆、宜靜不宜動。

## 含義

- 主變動不安、事多反覆、宜靜不宜動
- 凡事難以穩定，易生突發變故
- 情緒起伏大，容易衝動行事

## 起例方式

當四柱中**任意兩柱的地支構成六沖關係**，即為反吟。兩個互沖的柱位均受反吟影響。

### 六沖對照表

| 地支甲 | 地支乙 | 沖向 |
|--------|--------|------|
| 子 | 午 | 子午相沖 |
| 丑 | 未 | 丑未相沖 |
| 寅 | 申 | 寅申相沖 |
| 卯 | 酉 | 卯酉相沖 |
| 辰 | 戌 | 辰戌相沖 |
| 巳 | 亥 | 巳亥相沖 |

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 甲   | **子** |
| 月柱 | 丙   | 卯   |
| 日柱 | 戊   | **午** |
| 時柱 | 庚   | 申   |

上例中，年柱「子」與日柱「午」構成子午相沖，年柱與日柱均帶反吟。

## 各柱反吟效應

| 反吟所在柱位 | 影響範疇 |
|-------------|---------|
| 年月柱反吟 | 早年動盪，祖業難守，父母緣薄 |
| 年日柱反吟 | 自身多遷移，婚姻有變故 |
| 月日柱反吟 | 事業起伏大，感情多波折 |
| 日時柱反吟 | 晚年不安，子女緣薄或多衝突 |

## 傳統解說

反吟之象，猶如人仰天長嘆，滿腹委屈無處訴說。命帶反吟者，凡事易生反覆，計劃難以貫徹到底。大運流年遇六沖地支時，反吟之力更強，此時宜靜守，切忌輕舉妄動，以免招致更大波瀾。
