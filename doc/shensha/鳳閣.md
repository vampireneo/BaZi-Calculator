# 鳳閣

## 定義

鳳閣為文才之吉星，主文采風流、氣質優雅。命帶鳳閣者，多具藝文才華，氣質出衆，利於從事文職或藝術工作。

## 性質

- **類型**：吉星
- **取法**：依月支起算

## 計算方式

以出生月支查對應地支，該地支所在柱即為鳳閣所在位置。

| 月支 | 鳳閣落支 |
|------|----------|
| 寅   | 巳       |
| 卯   | 午       |
| 辰   | 未       |
| 巳   | 申       |
| 午   | 酉       |
| 未   | 戌       |
| 申   | 亥       |
| 酉   | 子       |
| 戌   | 丑       |
| 亥   | 寅       |
| 子   | 卯       |
| 丑   | 辰       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 幼年才藝出衆，家學淵源深厚     |
| 月柱   | 職場重文才，適合文教藝術行業   |
| 日柱   | 配偶氣質優雅，婚姻重精神交流   |
| 時柱   | 晚年優雅，子女多文藝氣息       |

## 傳統解說

鳳閣象徵鳳凰之閣，為文采與氣質之象徵。命帶鳳閣者，天生具有文人氣質，擅長文學、藝術、音樂等領域，處事優雅從容。與文昌、天乙同柱，則文才更為出衆，仕途上以文職為宜。
