# 天煞

## 定義

天煞又稱「天殺」，依日支或年支所屬三合局起算，主天降之禍，屬飛來橫禍類凶煞。

## 類型

**凶煞**

## 含義

主意外災禍、須防飛來橫禍、宜謹慎。天煞來自天而降，難以預測與防範，命中帶此煞者，易遭突如其來的意外事故，須時刻保持警惕。

## 起例方法

依**日支**或**年支**所屬三合局查表：

| 日支／年支（三合局） | 天煞所在支 |
|----------------------|-----------|
| 申、子、辰（水局）   | 戌         |
| 寅、午、戌（火局）   | 辰         |
| 巳、酉、丑（金局）   | 丑         |
| 亥、卯、未（木局）   | 未         |

> 查法：以日支（或年支）找到所屬三合局，再對應右欄地支，若該地支出現於四柱中，即為帶天煞。

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 幼年易逢意外，家庭多變故 |
| 月柱 | 事業中途受挫，防職場意外 |
| 日柱 | 本身易遇橫禍，出行需謹慎 |
| 時柱 | 晚景不寧，防老年意外 |

## 傳統解釋

天煞為天降之殺，古籍認為此煞主人遭受天意難測之災，非人力所能預防。凡帶天煞者，宜廣積善德，以德化煞，平日多做公益慈善之事，可減輕煞氣影響。行事切忌冒險衝動，遇事三思而後行。
