# 羅睺

## 定義

羅睺為北斗隱星之一，屬陰邪詭秘之星。主陰謀詭計、暗中破害，須防小人作祟。

## 性質

- **類型**：凶星
- **取法**：依年支起算

## 計算方式

以出生年支查對應地支，該地支所在柱即為羅睺所在位置。

| 年支 | 羅睺落支 |
|------|----------|
| 子   | 巳       |
| 丑   | 午       |
| 寅   | 未       |
| 卯   | 申       |
| 辰   | 酉       |
| 巳   | 戌       |
| 午   | 亥       |
| 未   | 子       |
| 申   | 丑       |
| 酉   | 寅       |
| 戌   | 卯       |
| 亥   | 辰       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 祖業受損，家族多暗鬥           |
| 月柱   | 工作易受小人陷害，職場多是非   |
| 日柱   | 婚姻多猜疑，配偶關係複雜       |
| 時柱   | 子女緣薄，晚年防暗中之害       |

## 傳統解說

羅睺為北斗隱星，隱而不顯，象徵陰邪詭秘之事。命帶羅睺者，一生中易遭小人陷害，暗中破壞，諸事宜謹慎行事，避免輕信他人。與吉星同柱可化解部分凶性。
