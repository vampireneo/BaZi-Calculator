# 歸祿

## 定義

歸祿，又稱「祿歸時上」，是指日干所對應之祿神出現於**時柱地支**的神煞。祿者，福祿之意，主富貴榮華。歸祿取象晚年福報歸身，子孫賢孝，福祿圓滿。

**類型**：吉神

## 含義

- 主晚年富貴、生活安逸
- 子孫賢孝、家庭和睦
- 福祿歸身、晚景豐隆
- 事業有成、聲望良好

## 起例方法

以**日干**查對應地支，若該地支出現於**時柱**，即為歸祿。

| 日干 | 祿神（時支） |
|------|------------|
| 甲   | 寅         |
| 乙   | 卯         |
| 丙   | 巳         |
| 丁   | 午         |
| 戊   | 巳         |
| 己   | 午         |
| 庚   | 申         |
| 辛   | 酉         |
| 壬   | 亥         |
| 癸   | 子         |

**例**：日干為甲，時柱地支為寅，即為歸祿。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 時柱 | 歸祿本位，主晚年福祿，子孫賢孝，老來享福 |

> 歸祿專論時柱，故效應集中於晚年運勢與子嗣宮位。

## 傳統解說

《神峰通考》云：「時上一位貴，終身用不盡。」歸祿之人，祿神藏於時支，象徵福報歸藏於晚年，有積德厚福之象。其人多勤勉向善，中年積累，晚年自然福祿雙至。子孫宮旺，後代多有出息，能承繼家業，令父母安享晚年。命有歸祿者，宜積善修德，方能充分發揮此吉神之效。
