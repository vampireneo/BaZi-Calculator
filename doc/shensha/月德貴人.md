# 月德貴人

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 吉神 |
| 別名 | 月德 |
| 取用基準 | 月支（依三合局歸組） |

## 定義與含義

月德貴人為命盤中與天德貴人並列的重要吉神，象徵月之德氣的庇護與品德的積累。命帶月德貴人者，主品德高尚、處事平順、有福德庇佑，為人善良正直，做事有原則，能以德服人，一生中多有隱性的貴人相助。

## 起算方法

以**月支所屬的三合局**為基準，查看四柱天干中是否出現對應天干。

| 月支三合局 | 涵蓋月份 | 月德貴人天干 |
|-----------|---------|------------|
| 寅午戌（火局） | 寅月、午月、戌月 | 丙 |
| 申子辰（水局） | 申月、子月、辰月 | 壬 |
| 亥卯未（木局） | 亥月、卯月、未月 | 甲 |
| 巳酉丑（金局） | 巳月、酉月、丑月 | 庚 |

## 判斷步驟

1. 找出月支（月柱地支）
2. 判斷月支所屬三合局
3. 對照上表查出對應月德貴人天干
4. 在四柱的天干中搜尋該天干
5. 出現即代表命帶月德貴人

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 祖上品德高尚，家族有積善之名，幼年受惠 |
| 月柱 | 工作中以德取信，同事上司信賴，職場平順 |
| 日柱 | 個人品德受人肯定，配偶賢良，婚姻美滿 |
| 時柱 | 晚年福德圓滿，子女孝順賢德，晚景幸福 |

## 月德貴人特殊格局

| 格局 | 說明 |
|------|------|
| 天月二德同現 | 與天德貴人同時入命，化煞能力最強，一生吉祥 |
| 月德透干 | 月德天干出現在天干，力量最強，品德廣為人知 |
| 月德坐印 | 月德與印星同柱，學識與品德兼備，名望俱佳 |
| 月德制凶 | 能化解命盤中的凶神惡煞，降低災厄發生機率 |

## 月德與天德的比較

| 比較項目 | 天德貴人 | 月德貴人 |
|---------|---------|---------|
| 取用基準 | 月支對應干或支 | 月支三合局對應干 |
| 查找對象 | 天干或地支 | 僅天干 |
| 象徵意義 | 上天賜予的保護 | 自身德行的積累 |
| 化煞力度 | 較強 | 略弱於天德 |

## 傳統解說

月德貴人代表月之精華德氣，月有陰晴圓缺，象徵循環與滋養。古人認為，月德源自人自身積累的善行與品德，與天德的外在庇護相輔相成。

命帶月德貴人者，處事圓融，善解人意，不易與人結怨，即便遭遇困境，也能靠自身的品德與人脈化解。此神煞對女命尤為有利，主婦德賢淑，家庭和諧。

月德貴人的力量在於「潛移默化」，非急難時的即時救援，而是長期德行積累帶來的吉祥庇護。

> **注意**：月德貴人僅查天干，不查地支，這是與天德貴人（可查干支）的重要區別。兩者同現時，化煞效果最為顯著。
