# 大耗

## 定義

大耗為歲破之後一位地支，即十二地支循環中緊接歲破之下一支。大耗主財物嚴重耗損、破財敗業，其損傷程度較小耗更為嚴重，為命理中重要之財耗凶煞。

## 性質

**類型**：凶星

**主要含義**：主財物耗損、破財敗業、須防盜竊。大耗較小耗損傷更重，凡命中帶大耗，財來財去，難以積存，須防意外破財。

## 起例方法

以**年支**先查歲破，再取歲破之**後一位**地支（十二支順序循環），即為大耗。

| 年支 | 歲破 | 大耗（歲破後一位） |
|-----|------|-----------------|
| 子  | 午   | 未              |
| 丑  | 未   | 申              |
| 寅  | 申   | 酉              |
| 卯  | 酉   | 戌              |
| 辰  | 戌   | 亥              |
| 巳  | 亥   | 子              |
| 午  | 子   | 丑              |
| 未  | 丑   | 寅              |
| 申  | 寅   | 卯              |
| 酉  | 卯   | 辰              |
| 戌  | 辰   | 巳              |
| 亥  | 巳   | 午              |

> **推算說明**：十二地支順序為子→丑→寅→卯→辰→巳→午→未→申→酉→戌→亥，循環排列，歲破後一位即為大耗。

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 祖業難守，早年財運受損 |
| 月柱   | 事業財帛耗散，中年破敗 |
| 日柱   | 個人財物損耗嚴重，婚姻亦受牽連 |
| 時柱   | 晚年財帛難聚，須防子女耗財 |

## 傳統解說

大耗為財星剋破之象，傳統命理視之為嚴重財耗之凶神。命中逢大耗，主財物無故耗散，或遭盜竊，或投資失利，或因官訟破財。大耗與歲破同柱，則破敗尤重。流年逢大耗，宜保守理財，切勿輕易投資冒險。
