# 計都

## 定義

計都為北斗隱星之一，主計謀多端、心機深沉。雖具深謀遠慮之才，然帶有陰險之性，須防陰謀暗算。

## 性質

- **類型**：凶星
- **取法**：依年支起算

## 計算方式

以出生年支查對應地支，該地支所在柱即為計都所在位置。

| 年支 | 計都落支 |
|------|----------|
| 子   | 亥       |
| 丑   | 子       |
| 寅   | 丑       |
| 卯   | 寅       |
| 辰   | 卯       |
| 巳   | 辰       |
| 午   | 巳       |
| 未   | 午       |
| 申   | 未       |

| 戌   | 酉       |
| 亥   | 戌       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 家族多暗鬥，祖輩多心機之人     |
| 月柱   | 職場善謀略，然易招忌恨         |
| 日柱   | 配偶心機深，婚姻多猜疑         |
| 時柱   | 子女聰慧但個性陰沉，晚景多慮   |

## 傳統解說

計都與羅睺並列北斗隱星，主深謀遠慮之象。命帶計都者，思慮縝密，善於謀略，然陰險之性難免，易涉陰謀之事。宜善用智慧於正途，避免以謀害人，否則招禍上身。
