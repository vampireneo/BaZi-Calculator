# 紫微

## 定義

紫微為帝星，屬尊貴顯赫之吉星，主權威在握、仕途順遂。命帶紫微者，一生多貴氣，易居要職。

## 性質

- **類型**：吉星
- **取法**：依年支起算

## 計算方式

以出生年支查對應地支，該地支所在柱即為紫微所在位置。

| 年支 | 紫微落支 |
|------|----------|
| 子   | 卯       |
| 丑   | 寅       |
| 寅   | 丑       |
| 卯   | 子       |
| 辰   | 亥       |
| 巳   | 戌       |
| 午   | 酉       |
| 未   | 申       |
| 申   | 未       |
| 酉   | 午       |
| 戌   | 巳       |
| 亥   | 辰       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 出身貴族，祖蔭深厚，早年顯貴   |
| 月柱   | 事業順遂，易獲上司賞識提拔     |
| 日柱   | 配偶尊貴，婚姻有助提升地位     |
| 時柱   | 晚年顯達，子女有成，福壽雙全   |

## 傳統解說

紫微為北斗帝星，象徵至高無上的權威與尊貴。命帶紫微者，天生具有領導氣質，能得貴人提攜，仕途或事業上易居高位。與祿馬同柱，則富貴雙全之象更顯。為命局中最尊貴的吉星之一。
