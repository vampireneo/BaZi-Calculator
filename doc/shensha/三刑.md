# 三刑

## 定義

三刑（Sān Xíng）為八字地支神煞之一，屬**凶煞**。三刑為地支相剋之力，主刑傷災禍、六親不和，命帶三刑者宜謹慎行事，防官非、刑傷之禍。

## 含義

- 主刑傷災禍、意外血光
- 六親不和，易與家人、同事產生衝突
- 易涉及官非訴訟
- 嚴重時有牢獄之象

## 起法（以四柱地支查）

三刑分為三種類型，需在四柱地支中出現指定組合：

### 類型一：寅巳申三刑（恃勢之刑）

| 刑支 | 說明 |
|------|------|
| 寅、巳、申 | 三支皆需出現於四柱中方成三刑 |

- 寅刑巳、巳刑申、申刑寅，互相刑剋
- 主仗勢欺人、爭強好勝，易引發衝突

### 類型二：丑戌未三刑（無恩之刑）

| 刑支 | 說明 |
|------|------|
| 丑、戌、未 | 三支皆需出現於四柱中方成三刑 |

- 丑刑戌、戌刑未、未刑丑，互相刑剋
- 主忘恩負義、六親無情，易因財物起紛爭

### 類型三：子卯相刑（無禮之刑）

| 刑支 | 說明 |
|------|------|
| 子、卯 | 兩支同時出現於四柱中即成相刑 |

- 子刑卯、卯刑子，相互刑剋
- 主無禮、不知廉恥，易有桃色糾紛

## 傳統解說

三刑為地支中最具破壞力的組合之一。古籍云：「三刑入命，刑傷難免，六親緣薄，禍患纏身。」三刑之力源自地支五行的相剋關係，傷害力較六沖更為深遠，因其為暗中傷害，不易察覺。命帶三刑者宜從事法律、醫療、軍警等化煞為用的行業，方能將刑煞之力轉化為助力。
