# 官符

## 定義

官符為主官非訴訟之凶煞，「官」指官府衙門，「符」指符書文書，合意為官府文書之糾紛。凡命中帶官符，易與官府、法律、訴訟相關，須防口舌是非及法律糾紛。

## 性質

**類型**：凶星

**主要含義**：主官非訴訟、須防口舌是非、避免爭執。官符主官司口舌，流年太歲相刑沖更重，凡逢此煞宜謹言慎行，避免與人結怨。

## 起例方法

以**年支**查對應地支，凡四柱（月、日、時）地支中出現對應地支，即為命中帶官符。

| 年支 | 官符地支 |
|-----|---------|
| 子  | 卯      |
| 丑  | 辰      |
| 寅  | 巳      |
| 卯  | 午      |
| 辰  | 未      |
| 巳  | 申      |
| 午  | 酉      |
| 未  | 戌      |
| 申  | 亥      |
| 酉  | 子      |
| 戌  | 丑      |
| 亥  | 寅      |

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 祖上或幼年家中有官非，或早年易惹口舌 |
| 月柱   | 事業上易有官司糾紛，與上司關係需謹慎 |
| 日柱   | 本人易有法律訴訟，婚姻亦易生口角是非 |
| 時柱   | 晚年或子女易有官非，或因子女惹官司 |

## 傳統解說

官符在傳統命理中屬年神諸煞之一，專主官府訴訟及口舌是非之事。古籍云：「官符入命，官非難免。」凡流年太歲逢官符，尤其與命局中地支相刑相沖，則官非之象更為顯著。命中逢官符者，平日宜謹言慎行，合約文書宜仔細審閱，避免與人發生金錢財務糾紛，以防被牽連入官司。
