# 十惡大敗

## 定義

十惡大敗為八字神煞之一，屬**凶神**，主錢財難聚、事業多阻、須防破敗。此十日干支相剋且藏有凶煞，為命理中論財運、事業最忌之煞神。

## 含義

- 錢財難聚，財來財去
- 事業多阻，難以成就
- 辛苦打拼，成果易被破壞
- 須防投資失利、破財之事
- 行事宜謹慎，避免冒進

## 起法（以日柱干支判斷）

十惡大敗以**日柱干支**為準，以下十組干支日柱即帶此煞：

| 序號 | 日柱 | 五行特性 |
|------|------|----------|
| 1    | 甲辰 | 木剋土，辰中藏戊土受剋 |
| 2    | 乙巳 | 木逢巳火洩氣 |
| 3    | 丙申 | 火剋金，申中庚金受剋 |
| 4    | 丁亥 | 火逢亥水相剋 |
| 5    | 戊戌 | 土旺極易枯燥，難生萬物 |
| 6    | 己丑 | 土逢丑濕，陰濁之氣重 |
| 7    | 庚辰 | 金逢辰，辰中有癸水洩氣 |
| 8    | 辛巳 | 金逢巳火相剋 |
| 9    | 壬申 | 水逢申，雖有生助但金水氾濫 |
| 10   | 癸亥 | 水逢亥，水勢過旺失控 |

> 凡日柱干支屬以上十組之一，即命帶十惡大敗。

## 各柱影響

| 情況 | 影響 |
|------|------|
| 命帶此煞 | 財運起伏，事業難以持續穩定 |
| 命局有制化 | 若有印星、比劫化解，影響可減輕 |
| 流年遇此煞 | 當年謹防財務損失、事業受阻 |
| 大運逢此 | 該段大運須格外謹慎理財、守業 |

## 化解之道

- 宜穩健行事，不可貪圖暴利
- 避免大額投資或冒險生意
- 多積德行善，以德化煞
- 宜選擇踏實職業，以勤補拙

## 傳統解說

古籍云：「十惡大敗，行事多乖。」此十日天干地支相剋失和，五行之氣衝突激烈，故主破敗之象。然命理之道並非一成不變，若命局格局清純，或有吉神護持，十惡大敗之影響可大幅降低。有此煞者，宜勤勉謹慎，方能化解破敗之兆。
