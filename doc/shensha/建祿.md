# 建祿

## 定義

建祿為八字神煞之一，屬**吉神**，主自立成家、財祿豐盈、事業有成。建祿為祿神入月令，日干得令旺盛，為命理中論財祿事業最重要之格局之一。

## 含義

- 自立成家，不依賴他人
- 財祿豐盈，衣食無虞
- 事業有成，靠自身努力致富
- 獨立自主，性格剛強
- 晚年財祿穩固

## 起法（以日干查月支）

建祿以**日干**對應之祿神出現在**月支**為判斷條件：

| 日干 | 祿神（月支） | 說明 |
|------|------------|------|
| 甲   | 寅月       | 甲木之祿在寅 |
| 乙   | 卯月       | 乙木之祿在卯 |
| 丙   | 巳月       | 丙火之祿在巳 |
| 丁   | 午月       | 丁火之祿在午 |
| 戊   | 巳月       | 戊土之祿在巳 |
| 己   | 午月       | 己土之祿在午 |
| 庚   | 申月       | 庚金之祿在申 |
| 辛   | 酉月       | 辛金之祿在酉 |
| 壬   | 亥月       | 壬水之祿在亥 |
| 癸   | 子月       | 癸水之祿在子 |

> 判斷方式：查日干，再看月支是否為對應祿神地支，完全相符則命格為建祿格。

## 格局特性

| 情況 | 影響 |
|------|------|
| 建祿格清純 | 自力更生，事業財祿俱佳 |
| 建祿逢財官透出 | 格局更高，富貴雙全 |
| 建祿逢比劫重重 | 財祿易被分奪，宜防小人 |
| 建祿逢官煞剋制 | 日主受制，須以印星化解 |

## 建祿格之優缺點

| 優點 | 注意事項 |
|------|----------|
| 自立自強，財祿自來 | 忌見羊刃，恐成剛強過度 |
| 不依賴祖業，靠自身努力 | 建祿格不喜再見比劫分財 |
| 月令得祿，日主氣旺 | 須有財官配合方能富貴 |

## 傳統解說

建祿格為命理中十格之一，古籍云：「建祿居月令，自立門戶。」祿者，俸祿也，日主之祿神坐月令，表示日主在最重要之時令位置得到旺氣支持，猶如官員坐享俸祿，衣食豐足。

建祿格之人，天生具有自立自強之精神，不依賴祖蔭，憑藉個人努力開創事業。此格局之成就高低，端視命局中財星、官星是否透出配合，若三者齊備，則富貴可期。
