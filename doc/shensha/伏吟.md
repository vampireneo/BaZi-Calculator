# 伏吟

## 定義

伏吟為八字神煞之一，屬**凶煞**。「伏」者，伏地也；「吟」者，呻吟也。伏吟如人伏地呻吟，主諸事停滯不前，難有進展。

## 含義

- 主重複不順、事多反覆、難有進展
- 諸事如同陷入泥沼，進退兩難
- 情緒低落，容易陷入自怨自艾

## 起例方式

當四柱（年、月、日、時）中，**任意兩柱或以上出現相同的地支**，即構成伏吟。

### 計算規則

| 條件 | 說明 |
|------|------|
| 兩柱地支相同 | 構成伏吟 |
| 三柱地支相同 | 伏吟力量更重 |
| 四柱地支相同 | 極罕見，伏吟極重 |

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 甲   | **子** |
| 月柱 | 丙   | 午   |
| 日柱 | 戊   | **子** |
| 時柱 | 庚   | 申   |

上例中，年柱與日柱地支同為「子」，構成伏吟。

## 各柱伏吟效應

| 伏吟所在柱位 | 影響範疇 |
|-------------|---------|
| 年柱伏吟 | 祖業重複，家族多困頓 |
| 月柱伏吟 | 事業停滯，工作反覆不順 |
| 日柱伏吟 | 婚姻感情多波折，夫妻易反目 |
| 時柱伏吟 | 子女緣薄，晚年多操勞 |

## 傳統解說

伏吟之象，猶如人伏臥於地，發出呻吟之聲，無法起身前行。命帶伏吟者，凡事容易原地踏步，努力難以轉化為成果。尤以大運流年重逢伏吟地支時，諸事阻礙更甚，宜守不宜攻，待時機成熟再謀進取。
