# 黃幡

## 定義

黃幡為凶星之一，主喪服之事，與喪門性質相近。命帶黃幡者，須防孝服臨身，家宅不安，多有喪葬憂慮。

## 性質

- **類型**：凶星
- **取法**：依年支起算

## 計算方式

以出生年支查對應地支，該地支所在柱即為黃幡所在位置。

| 年支 | 黃幡落支 |
|------|----------|
| 子   | 未       |
| 丑   | 申       |
| 寅   | 酉       |
| 卯   | 戌       |
| 辰   | 亥       |
| 巳   | 子       |
| 午   | 丑       |
| 未   | 寅       |
| 申   | 卯       |
| 酉   | 辰       |
| 戌   | 巳       |
| 亥   | 午       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 幼年有喪親之憂，祖輩早逝       |
| 月柱   | 中年親長多病，須防服喪         |
| 日柱   | 配偶或本人體弱，家宅多喪事     |
| 時柱   | 晚年易有喪事，子女宜多關注健康 |

## 傳統解說

黃幡古時為喪禮所用之黃色旗幡，象徵喪葬之事。命帶黃幡者，一生中與喪事緣分較深，易逢親友離世之事。宜多積德行善，廣做善事，以化解其喪葬之氣。與喪門同柱，喪事應驗尤為明顯。
