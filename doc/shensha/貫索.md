# 貫索

## 定義

貫索（Guàn Suǒ）為八字神煞之一，屬**凶煞**。貫索如繩索纏身，主牢獄之象，命帶貫索者主牢獄之災、官非纏身，須防訴訟糾紛。

## 含義

- 主牢獄之災、官司纏身
- 易捲入法律糾紛或訴訟
- 行動受到束縛，自由受限
- 嚴重時有入獄或被拘押之象

## 起法（以年支查）

貫索以**出生年份地支**對應，取地支十二支順序中的下一支：

| 年支 | 貫索所在支 |
|------|-----------|
| 子年 | 丑 |
| 丑年 | 寅 |
| 寅年 | 卯 |
| 卯年 | 辰 |
| 辰年 | 巳 |
| 巳年 | 午 |
| 午年 | 未 |
| 未年 | 申 |
| 申年 | 酉 |
| 酉年 | 戌 |
| 戌年 | 亥 |
| 亥年 | 子 |

**查法**：以出生年份地支，對照上表找出對應地支。若該地支出現於年、月、日、時任一柱，即命中帶貫索。

## 各柱影響

| 所在柱 | 影響 |
|--------|------|
| 年柱 | 祖輩或童年曾有法律糾紛，家庭有官非 |
| 月柱 | 中年易有訴訟或事業上的法律問題 |
| 日柱 | 本身易捲入官非，行動自由受限 |
| 時柱 | 晚年或子女有牢獄、官非之象 |

## 傳統解說

貫索之名源自古代刑具，繩索貫串犯人之象。古籍云：「貫索加臨，身陷囹圄，官非難免，自由受縛。」命帶貫索者並不必然入獄，但一生中與法律、訴訟相關的事務較常人為多。行事宜守法守規，避免涉及灰色地帶，以免貫索之煞應驗。若命局有吉神化解，訴訟雖有，尚可化解。
