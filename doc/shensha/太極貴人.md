# 太極貴人

## 定義

太極貴人依日干查對應地支，為玄學術數領域最重要的吉星之一，主智慧超群與近貴得福。

## 類型

**吉星**

## 含義

主近貴得福、智慧超群、適合玄學術數研究。命中帶太極貴人者，思維深邃，對哲學、玄學、易學等深奧學問有天生的領悟力，且一生易得貴人相助，逢凶化吉。

## 起例方法

依**日干**查對應地支，若對應地支出現於四柱中即帶太極貴人：

| 日干     | 太極貴人地支 |
|----------|------------|
| 甲、乙   | 子、午      |
| 丙、丁   | 卯、酉      |
| 戊、己   | 辰、戌、丑、未 |
| 庚、辛   | 寅、亥      |
| 壬、癸   | 巳、申      |

> 注意：戊己日干對應四個地支（辰戌丑未），因土旺四季，貴人最多。

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 祖蔭深厚，幼年得長輩提攜 |
| 月柱 | 事業得貴人相助，職涯發展順利 |
| 日柱 | 本人智慧出眾，配偶亦為貴人 |
| 時柱 | 晚年德望日隆，子女出色 |

## 傳統解釋

太極貴人源於太極之道，象徵陰陽調和、智慧圓融之境界。古人認為此貴人為諸貴人之首，凡命中見太極貴人，主此人天賦聰慧，能參透玄機。尤其適合從事命理、風水、易學、佛道等玄學研究，以及哲學、數學、科學等需要深度思維之領域。此星多者，人生際遇玄妙，常有化險為夷之奇遇。
