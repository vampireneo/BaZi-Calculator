# 詞館

## 定義

詞館屬**吉星**，為八字神煞之一。主文采斐然、能言善辯、利於文職法律。命帶詞館者，具有優秀的語言與文字表達能力，善於組織思維並清晰陳述，在文學、法律、外交等領域有天賦優勢。

## 起例方法

以**日干**取之。

| 日干 | 詞館（地支） |
|:---:|:---:|
| 甲 | 寅 |
| 乙 | 丑 |
| 丙 | 申 |
| 丁 | 巳 |
| 戊 | 申 |
| 己 | 巳 |
| 庚 | 亥 |
| 辛 | 戌 |
| 壬 | 寅 |
| 癸 | 亥 |

**查法**：確認日干後，對照上表找出詞館之支。若命盤年、月、日、時任一柱地支與之相符，即為命帶詞館。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 幼年語言能力強，善於表達 |
| 月柱 | 職場中以口才文筆取勝 |
| 日柱 | 本身文采出眾，擅長溝通說服 |
| 時柱 | 晚年聲望高，或子女從事文職 |

## 傳統解釋

詞館古指翰林院詞臣供職之所，是古代文人政治仕途的最高殿堂。命帶詞館者，具備翰林之才，即能以文字服務國家與社會的能力。

現代命理中，詞館象徵語言文字的天賦，宜從事作家、律師、教師、記者、演說家、外交官等以言辭為業的職業。詞館與學堂同現，文學才華更為突出，常見於文壇名家或雄辯之士的命格中。
