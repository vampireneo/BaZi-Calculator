# 天廚貴人

## 定義

天廚貴人屬**吉星**，為八字神煞之一。主食祿豐厚、衣食無缺、生活富足。命帶天廚貴人者，一生衣食無憂，飲食享受佳，且多有人緣，常得他人款待與照顧。

## 起例方法

以**日干**取之。天廚貴人為日干之食神祿位所在地支。

| 日干 | 天廚貴人（地支） |
|:---:|:---:|
| 甲 | 巳 |
| 乙 | 午 |
| 丙 | 巳 |
| 丁 | 午 |
| 戊 | 申 |
| 己 | 酉 |
| 庚 | 亥 |
| 辛 | 子 |
| 壬 | 寅 |
| 癸 | 卯 |

**查法**：確認日干後，對照上表找出天廚貴人之支。若命盤年、月、日、時任一柱地支與之相符，即為命帶天廚貴人。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 家世有食祿，幼年衣食豐足 |
| 月柱 | 事業與財運佳，職場得助 |
| 日柱 | 本身飲食享受好，生活品質高 |
| 時柱 | 晚年衣食無憂，子女孝順奉養 |

## 傳統解釋

天廚貴人象徵上天賜予的「廚房」，取食神得祿之意。食神代表飲食、享受與才藝，其祿位旺盛則食祿源源不絕。

命帶此星者，多有烹飪、餐飲或服務業天賦，亦主口福佳、人緣好。在現代命理中，天廚貴人亦象徵從事飲食、餐飲、服務相關行業者容易成功，是論衣食財祿的重要吉星之一。
