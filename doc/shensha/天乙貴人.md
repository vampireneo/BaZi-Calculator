# 天乙貴人

## 基本資料

| 項目 | 內容 |
|------|------|
| 名稱 | 天乙貴人 |
| 類型 | 吉神 |
| 別名 | 天一貴人 |

## 定義與含義

天乙貴人為八字神煞中最重要的吉神之一，主一生貴人相助、逢凶化吉、遇難呈祥。凡命中有此神煞者，一生中遇到困難往往能得貴人幫扶，逢凶事亦能轉危為安。

天乙貴人代表來自高位者的庇護與提攜，有助於事業、官運、人際關係各方面的發展。多數旺地，則更能發揮其吉祥特性。

## 計算方法

**以日干查詢，看四柱其餘各柱地支是否符合：**

| 日干 | 天乙貴人地支 |
|------|-------------|
| 甲、戊、庚 | 丑、未 |
| 乙、己 | 子、申 |
| 丙、丁 | 亥、酉 |
| 辛 | 午、寅 |
| 壬、癸 | 卯、巳 |

**注意事項：**
- 天乙貴人不居子午，即子年、午年生人無天乙貴人（陰陽絕地）
- 以日干為主，找年柱、月柱、時柱地支中是否出現對應地支

## 影響分析

- **入命强旺**：事業順遂，貴人不斷，名利雙收
- **天乙合日**：貴人親自提攜，效果最強
- **逢合**：天乙被合住，貴人可能有所保留，助力稍減
- **逢沖剋**：貴人助力被削弱，需自身努力

## 傳統解說

《神峰通考》載：「天乙貴人，甲戊庚在丑未，乙己鼠猴鄉，丙丁豬雞位，壬癸兔蛇藏，六辛逢馬虎，此是貴人方。」

天乙乃十二神中最尊貴者，主掌人間大事，凡命中有天乙貴人落陷旺地，則其人一生受神明護佑，衣食自來。
