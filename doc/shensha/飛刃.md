# 飛刃

## 定義

飛刃為羊刃之對沖星，取羊刃所在地支之六沖。飛刃之力量比羊刃更為凶猛，主血光意外、刀傷手術，須格外謹慎。

## 性質

- **類型**：凶星
- **取法**：依日干起算（僅陽干有飛刃）

## 計算方式

飛刃為羊刃地支之六沖（對沖）。僅陽干（甲、丙、戊、庚、壬）具有羊刃與飛刃，陰干無此二星。

| 日干   | 羊刃 | 飛刃（羊刃六沖） |
|--------|------|-----------------|
| 甲（陽木）| 卯   | 酉              |
| 丙（陽火）| 午   | 子              |
| 戊（陽土）| 午   | 子              |
| 庚（陽金）| 酉   | 卯              |
| 壬（陽水）| 子   | 午              |
| 乙、丁、己、辛、癸 | 無   | 無  |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 幼年易有意外，家族有血光之事   |
| 月柱   | 職場有危險，易有手術刀傷       |
| 日柱   | 本人易受傷，婚姻多衝突磨擦     |
| 時柱   | 晚年須防意外，子女宜注意安全   |

## 傳統解說

飛刃與羊刃相沖，其力量更勝羊刃，為命局中最凶猛的煞星之一。命帶飛刃者，須注意血光之災，避免從事高危行業，手術、意外傷害之應驗較強。若命局五行有制化，或逢吉星，可適度化解其凶性。醫護、軍警人員帶此星，反可借其力成就事業。
