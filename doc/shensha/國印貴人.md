# 國印貴人

## 定義

國印貴人屬**吉星**，為八字神煞之一。主掌印信權柄、適合公職、有官運。命帶國印貴人者，多有機會進入政府機關或大型組織任職，具備管理才能，易獲授權與重任。

## 起例方法

以**日干**取之。

| 日干 | 國印貴人（地支） |
|:---:|:---:|
| 甲 | 戌 |
| 乙 | 亥 |
| 丙 | 丑 |
| 丁 | 寅 |
| 戊 | 丑 |
| 己 | 寅 |
| 庚 | 辰 |
| 辛 | 巳 |
| 壬 | 未 |
| 癸 | 申 |

**查法**：確認日干後，對照上表找出國印貴人之支。若命盤年、月、日、時任一柱地支與之相符，即為命帶國印貴人。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 家族有公職背景，幼年受官家庇蔭 |
| 月柱 | 事業仕途順遂，易獲上司賞識 |
| 日柱 | 本身掌權能力強，適合領導職 |
| 時柱 | 晚年仍有職務，或子女從事公職 |

## 傳統解釋

國印貴人取「國家印信」之意，象徵朝廷授予官員的印璽，是古代最高權威的象徵。命帶此星者，具備承擔公共責任的格局，適合從事政府、法律、教育、軍警等需要公信力的職業。

現代命理中，國印貴人亦論個人的信譽與公信力，帶此星者往往言而有信，深受社會信任，是論官職與權威的重要吉星。
