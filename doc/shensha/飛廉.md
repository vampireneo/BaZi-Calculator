# 飛廉

## 定義

飛廉（Fēi Lián）為八字神煞之一，屬**凶煞**。飛廉乃孤虛之神，主奔波勞苦，命帶飛廉者主東奔西走、勞碌不安，一生難得清閒安寧。

## 含義

- 主奔波勞碌、東奔西走
- 生活難以安定，居無定所
- 為事業或生計不得不四處奔波
- 難得安寧，心身俱疲

## 起法（以年支查）

飛廉以**出生年份地支**對應，查表得飛廉所在地支：

| 年支 | 飛廉所在支 |
|------|-----------|
| 子年 | 酉 |
| 丑年 | 戌 |
| 寅年 | 亥 |
| 卯年 | 子 |
| 辰年 | 丑 |
| 巳年 | 寅 |
| 午年 | 卯 |
| 未年 | 辰 |
| 申年 | 巳 |
| 酉年 | 午 |
| 戌年 | 未 |
| 亥年 | 申 |

**查法**：以出生年份地支，對照上表找出對應地支。若該地支出現於年、月、日、時任一柱，即命中帶飛廉。

## 各柱影響

| 所在柱 | 影響 |
|--------|------|
| 年柱 | 童年時期居所不定，祖籍易遠離 |
| 月柱 | 中年事業奔波，工作多變動 |
| 日柱 | 本身勞碌命，難以安居樂業 |
| 時柱 | 晚年奔波，或為子女操勞奔走 |

## 傳統解說

飛廉乃孤虛之神，形如飛鳥，止不得息。古籍云：「飛廉入命，身如飛廉，東奔西走，難得安居。」命帶飛廉者一生勞碌，鮮少安閒，雖辛苦奔波，但若能善用此星主動、靈活之特性，從事需要頻繁移動的行業（如業務、旅遊、運輸等），反能以勞換財，化煞為用。切忌好逸惡勞，順應飛廉奔波之性，方能趨吉避凶。
