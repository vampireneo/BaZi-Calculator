# 沐浴

## 定義

沐浴，為十二長生之第二位，又稱「敗地」。取象嬰兒初生，以水沐浴去垢，象徵人生初始之脆弱與純淨。沐浴之地帶有桃花氣息，多主感情風流之事。

**類型**：中性（具吉凶雙重性）

## 含義

- 主風流多情、桃花旺盛
- 感情豐富、異性緣佳
- 須防感情糾紛、桃花劫
- 沐浴又稱敗地，乃沐浴去垢之象，主人多才藝但易放蕩

## 起例方法

以**日干**查對應沐浴地支，若該地支出現於四柱之中，即為沐浴。

| 日干 | 沐浴位 |
|------|------|
| 甲   | 子   |
| 乙   | 巳   |
| 丙   | 卯   |
| 丁   | 申   |
| 戊   | 卯   |
| 己   | 申   |
| 庚   | 午   |
| 辛   | 亥   |
| 壬   | 酉   |
| 癸   | 寅   |

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 年柱 | 幼年環境較不穩定，祖業難守，早年多風波 |
| 月柱 | 事業起伏，感情與工作易糾纏，人際複雜 |
| 日柱 | 婚姻感情多波折，配偶桃花旺，感情易生變 |
| 時柱 | 晚年仍桃花不斷，子女感情事多，老來不寂寞 |

## 傳統解說

《三命通會》云：「沐浴者，如人出生，去垢洗髓，清潔無瑕，然亦易受外誘。」沐浴之地本為桃花地，命帶沐浴者，天生具有吸引異性之魅力，風度翩翩或嫵媚動人。然此星帶有「敗」氣，若不能自持，易因感情而敗德喪業。命局若有正官、正印制化，可化桃花為才藝，反成一技之長。宜修身養性，以才情代替風流，方能吉中化凶。
