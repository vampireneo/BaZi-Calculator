# 專祿

## 定義

專祿，是指日干所對應之祿神出現於**日柱地支**（即與日干同柱），形成干支一體、祿神專屬之格局。取象祿星專一守身，不分散於他處。

**類型**：吉神

## 含義

- 主專心致志、事業穩健
- 財祿專一、不喜投機冒險
- 守業有成、腳踏實地
- 個性執著，一旦認定目標則全力以赴

## 起例方法

以**日干**查對應地支，若該地支即為**日柱地支**（日干與日支同柱），即為專祿。

| 日干 | 祿神（日支） | 日柱範例 |
|------|------------|---------|
| 甲   | 寅         | 甲寅日   |
| 乙   | 卯         | 乙卯日   |
| 丙   | 巳         | 丙巳日   |
| 丁   | 午         | 丁午日   |
| 戊   | 巳         | 戊巳日   |
| 己   | 午         | 己午日   |
| 庚   | 申         | 庚申日   |
| 辛   | 酉         | 辛酉日   |
| 壬   | 亥         | 壬亥日   |
| 癸   | 子         | 癸子日   |

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 日柱 | 專祿本位，干支一氣，祿神自守，主自力更生，財祿豐厚 |

> 專祿以日柱為論，代表本人本身之福祿，力量專一而不散。

## 傳統解說

《三命通會》載：「日支坐祿，謂之專祿，主人穩重自持，不事浮誇。」專祿之象，猶如守財有道，祿神不流散於外，盡歸本身所有。命帶專祿之人，處事踏實，不喜投機取巧，宜從事穩定行業，憑己之力積累財富。雖不及正祿之顯赫，然守成有餘，一生衣食無缺，晚年安穩。宜忌：不宜行事過於保守，應適時開拓，方能更上層樓。
