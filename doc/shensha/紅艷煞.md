# 紅艷煞

## 定義

紅艷煞又稱「紅鸞煞」或「桃花煞」之變體，依日干查對應地支，主感情風流之事。

## 類型

**中性煞**（有吉有凶，視格局而定）

## 含義

主風流多情、異性緣佳、但須防感情糾葛。帶紅艷煞者天生魅力出眾，易招異性青睞，若格局佳則感情順遂，若格局雜亂則恐陷多角關係或感情糾紛。

## 起例方法

依**日干**查對應地支，若該地支出現於四柱中即帶紅艷煞：

| 日干 | 紅艷煞地支 |
|------|-----------|
| 甲   | 午         |
| 乙   | 申         |
| 丙   | 寅         |
| 丁   | 未         |
| 戊   | 辰         |
| 己   | 辰         |
| 庚   | 戌         |
| 辛   | 酉         |
| 壬   | 子         |
| 癸   | 申         |

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 早年感情豐富，青春四溢 |
| 月柱 | 工作環境中異性緣旺 |
| 日柱 | 婚姻中感情熾烈，但易生變 |
| 時柱 | 晚年仍桃花不斷，感情不易平靜 |

## 傳統解釋

紅艷煞古稱「紅艷」，因其色如紅花艷麗，象徵感情之事。男命見之，易有風流韻事；女命見之，魅力過人但感情多波折。此煞本為中性，關鍵在於八字整體格局，若日主有力且格局清純，則紅艷煞反為增添個人魅力之利器；若格局混雜，則須防感情困擾。
