# 福星貴人

## 定義

福星貴人屬**吉星**，為八字神煞之一。主福氣臨門、一生平安、遇事有救。命帶福星貴人者，人生中常逢貴人相助，危難之時有人伸出援手，整體運勢平穩祥和。

## 起例方法

以**日干**取之。

| 日干 | 福星貴人（地支） |
|:---:|:---:|
| 甲 | 寅 |
| 乙 | 丑 |
| 丙 | 亥 |
| 丁 | 戌 |
| 戊 | 申 |
| 己 | 未 |
| 庚 | 巳 |
| 辛 | 辰 |
| 壬 | 寅 |
| 癸 | 丑 |

**查法**：確認日干後，對照上表找出福星貴人之支。若命盤年、月、日、時任一柱地支與之相符，即為命帶福星貴人。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 家族蔭福深厚，祖上積德 |
| 月柱 | 社會人際關係佳，事業多助力 |
| 日柱 | 本身福德深厚，婚姻美滿 |
| 時柱 | 晚年福報豐盈，子孫賢孝 |

## 傳統解釋

福星貴人為天上福德之星降臨命中之象，是八字神煞中論整體福氣與貴人緣分最重要的吉星之一。

命帶此星者，待人寬厚，廣結善緣，自然福氣相隨。古人認為此星與個人積德行善密切相關：前世或現世之善行，召來福星護佑。現代命理師亦常以此星論斷一個人的整體運氣底蘊，福星旺者，縱使遭遇困難，亦能逢凶化吉。
