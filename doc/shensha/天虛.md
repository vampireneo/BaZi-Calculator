# 天虛

## 定義

天虛為凶星之一，主虛耗不實、事多落空。命帶此星者，諸事難以落實成形，計劃易化為泡影。

## 性質

- **類型**：凶星
- **取法**：依年支起算

## 計算方式

以出生年支查對應地支，該地支所在柱即為天虛所在位置。

| 年支 | 天虛落支 |
|------|----------|
| 子   | 未       |
| 丑   | 申       |
| 寅   | 酉       |
| 卯   | 戌       |
| 辰   | 亥       |
| 巳   | 子       |
| 午   | 丑       |
| 未   | 寅       |
| 申   | 卯       |
| 酉   | 辰       |
| 戌   | 巳       |
| 亥   | 午       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 祖業難守，家族根基空虛         |
| 月柱   | 事業多虛耗，錢財難以積聚       |
| 日柱   | 婚姻名存實亡，感情關係空洞     |
| 時柱   | 子女緣薄，晚年計劃難以實現     |

## 傳統解說

天虛象徵虛空無實，主諸事不落地之象。命帶天虛者，常有好計劃卻難以執行，努力付出卻收效甚微。宜腳踏實地，避免好高騖遠，多務實行事，方可減輕其虛耗之性。
