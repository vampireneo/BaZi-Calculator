# 紅鸞

## 定義

紅鸞為八字神煞之一，屬**吉神**，主姻緣和合、喜事連連、利於婚嫁。為命理中論婚姻感情最重要的神煞之一。

## 含義

- 主姻緣和合，感情順利
- 喜事連連，婚嫁吉祥
- 人緣佳，異性緣旺
- 有桃花之意，主情感美滿

## 起法（以年支查）

紅鸞依**年支**起算，查對應地支即為紅鸞所在：

| 年支 | 紅鸞 |
|------|------|
| 子   | 卯   |
| 丑   | 寅   |
| 寅   | 丑   |
| 卯   | 子   |
| 辰   | 亥   |
| 巳   | 戌   |
| 午   | 酉   |
| 未   | 申   |
| 申   | 未   |
| 酉   | 午   |
| 戌   | 巳   |
| 亥   | 辰   |

> 若命局月、日、時支中出現紅鸞，或流年地支逢紅鸞，皆主婚嫁姻緣之事。

## 各柱影響

| 所在柱位 | 影響 |
|----------|------|
| 年柱     | 早年有姻緣，家族婚喜多 |
| 月柱     | 青年時期感情順遂，婚嫁吉利 |
| 日柱     | 婚姻美滿，夫妻感情和諧 |
| 時柱     | 晚年再逢喜緣，子女婚事佳 |

## 與天喜之關係

紅鸞與**天喜**互為對沖，兩者在十二地支中相差六位。

- **紅鸞**：主婚姻、姻緣、感情的締結
- **天喜**：主喜慶、添丁、財喜之事

兩者同現命局，或流年同時逢紅鸞天喜，婚嫁喜慶之事最為應驗，為論婚姻之重要指標。

## 傳統解說

紅鸞星源自「紅鸞星動」之說，古人以紅鸞為主婚之星。命局中若紅鸞坐旺位，或與日主形成合局，多主婚姻美滿、感情順遂。流年逢紅鸞，未婚者易得良緣，已婚者夫妻感情更加深厚，亦主喜慶之事。
