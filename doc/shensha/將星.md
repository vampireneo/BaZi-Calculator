# 將星

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 吉神 |
| 別名 | 將星貴人 |
| 取用基準 | 日支或年支所屬三合局 |

## 定義與含義

將星為命盤中象徵領導權威與統御能力的吉神。命帶將星者，主領導能力強、有權威、適合管理職位，天生具備指揮統領的氣魄，能夠服眾，適合擔任主管、領袖、軍政要職等需要決策與管理的崗位。

## 起算方法

以**日支**或**年支**所屬的三合局為基準，查看四柱地支中是否出現對應地支。

| 日支或年支三合局 | 將星地支 |
|----------------|---------|
| 申子辰（水局） | 子 |
| 寅午戌（火局） | 午 |
| 巳酉丑（金局） | 酉 |
| 亥卯未（木局） | 卯 |

## 判斷步驟

1. 找出日支（日柱地支）或年支（年柱地支）
2. 判斷其所屬三合局
3. 對照上表查出對應將星地支
4. 在四柱地支中搜尋該地支，出現即為將星

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 祖上有威望，或家族出現領袖人物 |
| 月柱 | 職場領導力強，易獲上司賞識，升遷有望 |
| 日柱 | 個人領袖氣質強，配偶有威嚴，婚姻中易主導 |
| 時柱 | 晚年受人尊敬，子女出人頭地，有管理才能 |

## 將星特殊格局

| 格局 | 說明 |
|------|------|
| 將星帶官 | 領導才能得正官加持，仕途順遂，易居要職 |
| 將星帶印 | 有謀略有學識，智慧型領導人 |
| 將星逢沖 | 權威受挑戰，領導地位不穩，易有競爭對手 |
| 將星三現 | 領導慾望強烈，但需防獨斷獨行 |

## 傳統解說

將星源自古代軍事占星，代表統率三軍的大將之才。古人認為命帶將星者，天生具有將帥之命，擁有統御他人的能力與氣質。

在現代命理應用中，將星不限於軍政領域，凡需要領導管理的場合皆適用。將星入命且有官印相助，主事業上易居領導地位，部屬服從，決策力強。

將星最忌被沖破，若將星所在地支受沖，則領導地位不穩，易遭下屬反叛或競爭者挑戰。

> **注意**：將星為吉神，但需配合整體命盤格局，若日主過弱，雖有將星亦難發揮領導效能。
