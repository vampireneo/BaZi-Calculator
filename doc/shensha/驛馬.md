# 驛馬

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 中性神煞 |
| 別名 | 驛馬星 |
| 取用基準 | 日支或年支所屬三合局 |

## 定義與含義

驛馬為命盤中象徵奔波移動與變動的中性神煞。命帶驛馬者，主奔波勞碌、適合外出發展、有遷移變動之象，一生少靜多動，常有出行、搬遷、換工作等情況，適合從事旅遊、運輸、業務、外貿等需要四處奔走的行業。

## 起算方法

以**日支**或**年支**所屬的三合局為基準，查看四柱地支中是否出現對應地支。

| 日支或年支三合局 | 驛馬地支 |
|----------------|---------|
| 申子辰（水局） | 寅 |
| 寅午戌（火局） | 申 |
| 巳酉丑（金局） | 亥 |
| 亥卯未（木局） | 巳 |

## 判斷步驟

1. 找出日支（日柱地支）或年支（年柱地支）
2. 判斷其所屬三合局
3. 對照上表查出對應驛馬地支
4. 在四柱地支中搜尋該地支，出現即為驛馬

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 祖上漂泊，或幼年時家庭多遷移 |
| 月柱 | 職業多變動，工作地點常更換，易異地工作 |
| 日柱 | 個人好動喜動，配偶或自身奔波，婚後異地生活 |
| 時柱 | 晚年仍多奔波，子女離鄉發展 |

## 驛馬特殊格局

| 格局 | 說明 |
|------|------|
| 驛馬帶財 | 驛馬與財星同現，主在外地或奔波中得財 |
| 驛馬逢沖 | 驛馬被沖動，變動更頻繁，甚至有意外之行 |
| 驛馬逢合 | 驛馬被合住，變動受阻，難以成行 |
| 驛馬臨官 | 奔波中得官位，適合政府外派或跨國企業 |

## 傳統解說

驛馬古稱傳遞公文之馬，象徵快速流動與訊息傳遞。古代驛馬入命，多主官差調遣、四方奔波。現代命理中，驛馬代表移動性強、不安於室，是從事業務、外交、航運、旅遊等行業的有利神煞。

命帶驛馬者，若能順勢而為、積極外出拓展，則可在異地或流動性工作中獲得成功；若逆勢守成，則易感到鬱悶不得志。

> **注意**：驛馬為中性神煞，奔波是否帶來吉凶，需視驛馬所臨天干的喜忌而定。
