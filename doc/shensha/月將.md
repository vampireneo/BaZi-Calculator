# 月將

## 定義

月將為月建之將神，主領導統御之象。命帶月將者，具統領才能，善於管理，易居要位，權威在握。

## 性質

- **類型**：吉星
- **取法**：依月支起算

## 計算方式

以出生月支查對應地支，該地支所在柱即為月將所在位置。

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 祖輩具領導地位，早年受人敬重   |
| 月柱   | 事業上善於管理，易升遷主管     |
| 日柱   | 配偶具領導才能，家中具主導地位 |
| 時柱   | 晚年仍具威望，子女多為領導人才 |

## 傳統解說

月將源自月建之將神，代表統領與號令之力。命帶月將者，天生具有組織管理能力，能統率衆人，適合擔任領導職位。與紫微、天乙同柱，則領導氣質更為顯著，仕途多有建樹。
