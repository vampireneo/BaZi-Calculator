# 天喜

## 定義

天喜為八字神煞之一，屬**吉神**，主喜事臨門、婚姻美滿、添丁進財。凡命局或流年遇天喜，多主喜慶之事接連而來。

## 含義

- 主喜事臨門，吉慶連綿
- 婚姻美滿，夫妻和諧
- 添丁進財，家運興旺
- 人緣極佳，廣受歡迎

## 起法（以年支查）

天喜依**年支**起算，查對應地支即為天喜所在：

| 年支 | 天喜 |
|------|------|
| 子   | 酉   |
| 丑   | 申   |
| 寅   | 未   |
| 卯   | 午   |
| 辰   | 巳   |
| 巳   | 辰   |
| 午   | 卯   |
| 未   | 寅   |
| 申   | 丑   |
| 酉   | 子   |
| 戌   | 亥   |
| 亥   | 戌   |

> 若命局月、日、時支中出現天喜，或流年地支逢天喜，皆主喜事降臨。

## 各柱影響

| 所在柱位 | 影響 |
|----------|------|
| 年柱     | 祖上有喜慶，家族興旺 |
| 月柱     | 事業、兄弟方面多喜事 |
| 日柱     | 婚姻美滿，配偶帶來喜事 |
| 時柱     | 子女緣佳，晚年喜慶 |

## 與紅鸞之關係

天喜與**紅鸞**互為對沖，兩者相差六位。紅鸞主婚姻、姻緣和合；天喜主喜慶、添丁、財喜。兩者合稱「紅鸞天喜」，同現命局或流年時，婚嫁喜事之應驗最為明顯。

## 傳統解說

《神煞賦》云：「天喜臨門，百事皆宜。」古人認為天喜為天上降喜之神，凡逢天喜之年，宜嫁娶、生育、開業、喬遷，諸事皆可順遂。流年大運逢天喜，更主婚喜或子嗣之喜。
