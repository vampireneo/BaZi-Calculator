# 學堂

## 定義

學堂屬**吉星**，為八字神煞之一。主聰明好學、學業有成、文采出眾。命帶學堂者，天資聰穎，學習能力強，對知識有天然的熱情與吸收力，在學術或文化領域多有成就。

## 起例方法

以**日干**取之。學堂為日干五行長生之位。

| 日干 | 學堂（地支） |
|:---:|:---:|
| 甲 | 亥 |
| 乙 | 午 |
| 丙 | 寅 |
| 丁 | 酉 |
| 戊 | 寅 |
| 己 | 酉 |
| 庚 | 巳 |
| 辛 | 子 |
| 壬 | 申 |
| 癸 | 卯 |

**查法**：確認日干後，對照上表找出學堂之支。若命盤年、月、日、時任一柱地支與之相符，即為命帶學堂。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 幼年聰慧，早期教育良好 |
| 月柱 | 青年學業優秀，進入良好學府 |
| 日柱 | 終身好學，知識廣博 |
| 時柱 | 晚年仍鑽研學問，或子女學業出色 |

## 傳統解釋

學堂取五行長生之位，長生象徵生命力最旺盛的起點，用於學問則代表學習之根基深厚、生生不息。命帶學堂者，往往在學生時期表現優異，且終身保持學習熱情。

現代命理中，學堂亦論一個人的智識能力與學習環境。宜從事教育、研究、寫作、法律、諮詢等需要深厚知識底蘊的職業，學堂旺者，常成為所在領域的專家或學者。
