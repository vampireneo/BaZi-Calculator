# 歲刑

## 定義

歲刑是依據年支與其他柱支之間的「三刑」關係所形成的神煞。凡命局中非年柱的地支，與年支構成刑沖關係者，即稱為歲刑。

**類型**：凶
**主要含義**：主刑傷災禍、須防意外、宜謹慎。歲刑主年份流年刑沖的影響，逢之須提防法律糾紛、身體傷損或人際衝突。

---

## 計算方法

依照**三刑理論**，各年支所對應的刑支如下：

| 年支 | 刑支 | 刑型 |
|------|------|------|
| 子年 | 卯 | 相刑 |
| 丑年 | 戌、未 | 三刑 |
| 寅年 | 巳、申 | 三刑 |
| 卯年 | 子 | 相刑 |
| 辰年 | 辰 | 自刑 |
| 巳年 | 寅、申 | 三刑 |
| 午年 | 午 | 自刑 |
| 未年 | 丑、戌 | 三刑 |
| 申年 | 寅、巳 | 三刑 |
| 酉年 | 酉 | 自刑 |
| 戌年 | 丑、未 | 三刑 |
| 亥年 | 亥 | 自刑 |

**判斷方式**：找出年支對應的刑支，若該刑支出現在月柱、日柱或時柱之地支，即構成歲刑。

---

## 各刑型說明

| 刑型 | 涵蓋地支 | 特性 |
|------|----------|------|
| 三刑 | 寅巳申、丑戌未 | 恃勢之刑，主權威衝突 |
| 相刑 | 子卯 | 無禮之刑，主禮法糾紛 |
| 自刑 | 辰午酉亥 | 自我損傷，主內耗暗病 |

---

## 各位置影響

| 刑支所在 | 影響重點 |
|----------|----------|
| 月柱 | 事業、兄弟、中年易有刑傷 |
| 日柱 | 配偶、本身，婚姻或健康易受損 |
| 時柱 | 子女、晚年，須防意外或官訟 |

---

## 傳統解說

《三命通會》云：「刑者，刑戮之義，主災厄、疾病、口舌、官非。」歲刑以年支為基準，代表流年與本命之間的刑剋作用。命帶歲刑者，逢流年引動時，易招法律糾紛、手術開刀或人際衝突，尤以三刑最為嚴重。自刑則多主自我傷害或暗疾纏身，宜謹慎處事，凡事以和為貴。
