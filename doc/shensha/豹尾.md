# 豹尾

## 定義

豹尾為凶星之一，性烈如豹，主凶惡暴戾、易有血光之災，須防意外衝突與傷害。

## 性質

- **類型**：凶星
- **取法**：依年支起算（歲破後二位）

## 計算方式

豹尾位於歲破地支後兩位（依地支十二順序計算）。以出生年支查下表：

| 年支 | 歲破 | 豹尾落支 |
|------|------|----------|
| 子   | 午   | 申       |
| 丑   | 未   | 酉       |
| 寅   | 申   | 戌       |
| 卯   | 酉   | 亥       |
| 辰   | 戌   | 子       |
| 巳   | 亥   | 丑       |
| 午   | 子   | 寅       |
| 未   | 丑   | 卯       |
| 申   | 寅   | 辰       |
| 酉   | 卯   | 巳       |
| 戌   | 辰   | 午       |
| 亥   | 巳   | 未       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 幼年多意外，家族易有血光之事   |
| 月柱   | 職場環境危險，工作易有衝突傷害 |
| 日柱   | 本人易遭血光，婚姻多爭鬥       |
| 時柱   | 晚年須防意外，子女易有危險     |

## 傳統解說

豹尾象徵豹之尾巴，兇猛急烈，主意外衝突與血光之象。命帶豹尾者，行事宜謹慎，避免涉足危險場所，注意交通安全及利器傷害。與羊刃同柱，則血光之應驗更為強烈。
