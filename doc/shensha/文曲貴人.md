# 文曲貴人

## 定義

文曲貴人為文采之星，主聰明才智、才思敏捷，與文昌貴人相輔相成，同為科舉文章之吉星。命中帶文曲者，多具文學藝術天賦，利於學業考試及文職發展。

## 性質

**類型**：吉星

**主要含義**：主文采出眾、才思敏捷、利於科舉考試。文曲為文采之星，與文昌相輔相成，凡命中見之，文章詩詞皆有出眾之能。

## 起例方法

以**日干**查對應地支，凡四柱（年、月、日、時）地支中出現對應地支，即為命中帶文曲貴人。

| 日干 | 文曲貴人地支 |
|------|------------|
| 甲   | 巳         |
| 乙   | 午         |
| 丙   | 申         |
| 丁   | 酉         |
| 戊   | 申         |
| 己   | 酉         |
| 庚   | 亥         |
| 辛   | 子         |
| 壬   | 寅         |
| 癸   | 卯         |

> **注**：文曲貴人之對應地支與文昌貴人相同，在多數傳統命書中兩者並列，意義略有側重，文昌主科名，文曲主文采。

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 祖上有文名，幼年聰慧好學 |
| 月柱   | 青年時期文才出眾，學業順遂 |
| 日柱   | 本人才華橫溢，文章秀麗 |
| 時柱   | 晚年文名顯達，或子女多才 |

## 傳統解說

文曲星源自北斗七星之第四星，古代命理以文曲主文章才藝。凡命中逢文曲，多聰慧好學，擅長文字表達，尤利於文科考試、寫作創作、教育傳播等領域。與文昌同現，則文名更著，主大貴顯達之命。
