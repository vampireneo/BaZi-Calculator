# 六厄

## 定義

六厄為八字神煞之一，屬**凶煞**。「六厄」意指六種厄難之神，主命主一生中災厄連連、困難重重，各種障礙接連而至，須謹慎應對。

## 含義

- 主災厄連連、困難重重
- 各種阻礙接踵而至，處境艱難
- 須有耐心面對逆境，不可輕易放棄
- 宜積德行善，以化解厄運

## 起算方式

以**年支**查六厄所在之地支，若該地支出現於命盤其他柱位，即為六厄。

### 對照表

| 年支 | 六厄位置 |
|------|----------|
| 子   | 卯       |
| 丑   | 辰       |
| 寅   | 巳       |
| 卯   | 午       |
| 辰   | 未       |
| 巳   | 申       |
| 午   | 酉       |
| 未   | 戌       |
| 申   | 亥       |
| 酉   | 子       |
| 戌   | 丑       |
| 亥   | 寅       |

> 規律：六厄所在地支為年支順推四位，與年支呈四位相隔之關係。

## 各柱位影響

| 柱位 | 影響說明 |
|------|----------|
| 年柱 | 祖業多厄，幼年環境困苦 |
| 月柱 | 事業多阻，中年遭遇較多厄難 |
| 日柱 | 個人多經歷逆境，婚姻亦有障礙 |
| 時柱 | 晚年仍有厄運，子女緣分多阻 |

## 傳統解說

六厄古稱六種厄難之神，代表命主在人生旅途中所面臨的種種障礙與困厄。命帶六厄者，往往需要比常人付出更多的努力，才能克服層層阻礙。然而，逆境亦是磨練意志、積累智慧的機遇。宜以積極心態面對厄難，廣結善緣、多行善事，藉助正能量化解負面影響。若六厄與吉神同柱，則可一定程度上減輕厄難的影響，困境中仍有轉機可尋。
