# 天赦

## 定義

天赦為八字神煞之一，屬**吉神**，主逢凶化吉、罪過可赦、貴人相助。為命理中化解凶煞、消災解厄最強力的吉神之一。

## 含義

- 逢凶化吉，化解災厄
- 罪過得赦，逢難呈祥
- 貴人相助，逢難有救
- 官司訴訟易獲寬宥
- 能解其他凶煞之力

## 起法（以出生月份季節配日柱）

天赦以**出生月支所屬季節**配合**日柱干支**判斷：

| 出生月（季節） | 月支範圍         | 天赦日柱 |
|----------------|------------------|----------|
| 春季           | 寅月、卯月       | 戊寅日   |
| 夏季           | 巳月、午月       | 甲午日   |
| 秋季           | 申月、酉月       | 戊申日   |
| 冬季           | 亥月、子月       | 甲子日   |
| 四季月         | 辰、戌、丑、未月 | 戊辰日   |

> 判斷方式：查出生之月支屬何季節，再看日柱是否為對應天赦日柱，完全相符則命帶天赦。

## 各柱影響

天赦以日柱干支為準，故主要影響本人一生之命運格局：

| 情況 | 影響 |
|------|------|
| 命帶天赦 | 一生逢難有貴人相救，化險為夷 |
| 流年逢天赦 | 當年諸事化凶為吉，是非消散 |
| 天赦逢凶煞 | 能有效化解凶煞之力，減輕災禍 |

## 傳統解說

天赦為上天赦免罪過之神。古籍記載，命帶天赦者，縱有過失，亦可得寬宥；縱遇凶煞，亦有化解之機。《協紀辨方書》認為天赦日為天地間最吉之日，萬事皆宜，尤利於求赦免、化解紛爭、逢難求助。

命帶天赦之人，往往在最危急之時，總有貴人出現相助，化解危難，故古人視天赦為護命之神。
