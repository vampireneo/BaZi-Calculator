# 天德貴人

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 吉神 |
| 別名 | 天德 |
| 取用基準 | 月支 |

## 定義與含義

天德貴人為命盤中最重要的吉神之一，象徵上天的庇佑與神聖的保護力量。命帶天德貴人者，主逢凶化吉、一生平安、有貴人扶持，能化解諸多災難與凶煞，是命盤中最具保護效果的神煞。

以**月支**（月柱地支）為基準，查看四柱天干與地支中是否出現對應的天干或地支。

## 判斷步驟

1. 找出月支（月柱地支）
2. 對照上表查出對應天德貴人（天干或地支）
3. 在四柱的天干與地支中搜尋該干支
4. 出現即代表命帶天德貴人

| 格局 | 說明 |

| 天德合日主 | 天德天干與日主相合，貴人與主命相合，吉力倍增 |

## 傳統解說

天德貴人源自古代術數，代表上天賦予的德氣庇護。古人認為，天有四德——元、亨、利、貞，對應四季流轉，天德貴人即承載此天地德氣的神煞，能化解命盤中的凶煞與災厄。

命帶天德貴人者，一生即便遭遇困難，也能逢凶化吉，死裡逃生，多有貴人在關鍵時刻出手相助。此神煞對命盤中的凶煞（如七殺、羊刃、天羅地網）有強力的化解作用。

天德貴人以「透干」（出現在天干）為力量最強，「藏支」（藏於地支藏干）次之。

> **注意**：天德貴人與月德貴人並稱「二德」，為命盤中最重要的化煞吉神，兩者同時入命則保護力極強。
