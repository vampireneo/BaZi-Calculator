# 指背

## 定義

指背為八字神煞之一，屬**凶煞**。指背主被人在背後指責批評，命帶此煞者易遭誹謗，須防小人從中作梗，宜謹言慎行以避口舌之非。

## 含義

- 主背後是非、易遭誹謗、須防小人
- 易被他人在背後議論批評
- 人際關係複雜，暗中有人作對

## 起例方式

依**年支**查對應地支，若該地支出現在命盤四柱中，即為帶指背。

> **注意**：指背與**浮沉**在此體系中使用相同的對照數值，但兩者性質與側重面不同：浮沉主事業起伏，指背主人際是非。

### 年支與指背對照表

| 年支 | 指背地支 |
|------|---------|
| 子   | 巳 |
| 丑   | 午 |
| 寅   | 未 |
| 卯   | 申 |
| 辰   | 酉 |
| 巳   | 戌 |
| 午   | 亥 |
| 未   | 子 |
| 申   | 丑 |
| 酉   | 寅 |
| 戌   | 卯 |
| 亥   | 辰 |

### 計算步驟

1. 取命盤年柱地支（年支）
2. 查上表得指背對應地支
3. 若月柱、日柱或時柱地支中有對應地支，即帶指背

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 壬   | **午**（查表：指背為亥）|
| 月柱 | 甲   | 子   |
| 日柱 | 丙   | 辰   |
| 時柱 | 戊   | **亥**（命中指背）|

年支「午」對應指背地支「亥」，時柱地支為「亥」，故帶指背。

## 各柱指背效應

| 指背所在柱位 | 影響範疇 |
|-------------|---------|
| 年柱 | 家族中易有背後是非，祖上名譽受損 |
| 月柱 | 職場中易遭同事暗算，上司有偏見 |
| 日柱 | 配偶或親密關係中易有猜疑與誹謗 |
| 時柱 | 晚年易遭子女或下屬背後議論 |

## 傳統解說

指背之象，猶如被人在背後指指點點，言語中傷，當事人往往渾然不知。命帶指背者，在人際交往中宜低調謙遜，少說多做，不輕易表露心跡，以減少被人議論的機會。尤其宜慎選朋友，遠離是非之地，方能保全名譽。
