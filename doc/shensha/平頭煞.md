# 平頭煞

## 定義

平頭煞為八字神煞之一，屬**凶煞**。「平頭」意指天干地支五行相剋、內外不協，主命主內外不和、干支矛盾，凡事易有內外衝突、表裡不一之象。

## 含義

- 主干支相剋、內外不和
- 表面與內心常有矛盾衝突
- 人際關係易生摩擦，合作多阻
- 宜加強溝通，化解內外分歧

## 起算方式

以**日柱（日干＋日支）**組合判定，以下20組特定干支組合即為平頭煞。

## 各柱位影響

| 柱位 | 影響說明 |
|------|----------|
| 年柱 | 祖業根基不穩，家庭內部不和 |
| 月柱 | 事業中內外矛盾多，上下關係緊張 |
| 日柱 | **本身為平頭煞**，性格內外矛盾最明顯 |
| 時柱 | 晚年子女關係不睦，晚景有礙 |

## 傳統解說

平頭煞之名取自天干地支五行相剋而「平齊相頂」之象，意指干支之氣互相對抗、難以協調。命主往往在表面行為與內心想法之間存在明顯落差，對外展現一面，對內卻有另一番想法，長期如此易造成心理壓力與人際困擾。宜透過修身養性、誠實待人來化解此煞，讓內外表裡趨於一致，方能減少矛盾衝突，走向平和順遂之人生。
