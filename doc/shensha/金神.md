# 金神

## 定義

金神（Jīn Shén）為八字神煞之一，屬**中性**（有制化則吉，無制則凶）。金神乃孤貴之星，性格剛毅，煞氣較重，須見火制化方為吉，否則反成凶象。

## 含義

- 主性格剛毅、意志堅強
- 有煞氣，若有制化則具權威與執行力
- 無火制化時，易性情偏激、帶來刀傷血光
- 適合武職、法律、軍警等需要煞氣之行業

## 起法（以時柱干支查）

金神以**特定時柱干支組合**論之，共三種組合：

| 時柱組合 | 說明 |
|---------|------|
| 乙丑 | 乙木坐丑，丑中含辛金，金神之一 |
| 己巳 | 己土坐巳，巳中含庚金，金神之一 |
| 癸酉 | 癸水坐酉，酉為純金，金神最旺 |

**查法**：以出生時辰對應之時柱干支，若符合上表三組之一，即命帶金神。

## 制化條件

| 情況 | 結果 |
|------|------|
| 命局有丙、丁火制金 | 金神化吉，有威望、執行力強 |
| 命局火運流年 | 制化金神，事業有成 |
| 命局無火制化 | 金神為凶，性情偏激，易生災禍 |
| 金神過旺又無制 | 刀傷、血光、意外之象 |

## 傳統解說

金神乃孤貴之星，煞中帶貴。古籍云：「金神入命，有制化者貴，無制化者凶。」金神帶有強烈的金屬肅殺之氣，需以火來熔煉制化，方能將其剛烈之性轉化為執行力與威望。命帶金神者若能得火制化，往往具備強大的意志力與執行力，適合從事需要魄力的工作；若無火制，則剛烈之氣難以收斂，易造成人際衝突或意外傷害。
