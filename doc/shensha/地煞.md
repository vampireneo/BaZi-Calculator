# 地煞

## 定義

地煞又稱「地殺」，依日支或年支所屬三合局起算，主地面傷害，屬跌傷摔傷類凶煞。

## 類型

**凶煞**

## 含義

主地面災害、須防跌傷摔傷、出行謹慎。地煞主傷害來自地面，如跌倒、摔傷、交通事故等，命中帶此煞者，行走間需格外小心，避免高處作業或危險地形。

## 起例方法

依**日支**或**年支**所屬三合局查表：

| 日支／年支（三合局） | 地煞所在支 |
|----------------------|-----------|
| 申、子、辰（水局）   | 辰         |
| 寅、午、戌（火局）   | 戌         |
| 巳、酉、丑（金局）   | 未         |
| 亥、卯、未（木局）   | 丑         |

> 查法：以日支（或年支）找到所屬三合局，再對應右欄地支，若該地支出現於四柱中，即為帶地煞。

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 幼年易跌傷，家中地面危險多 |
| 月柱 | 工作環境需注意安全，防職業傷害 |
| 日柱 | 本人四肢易受傷，出行謹慎 |
| 時柱 | 晚年行動不便，防老年跌傷 |

## 傳統解釋

地煞屬土氣過重或衝剋之性，古人認為此煞主人與大地相剋，行走間容易失足，尤其在陰雨路滑之時更須小心。帶地煞者宜避免攀高涉險，選擇工作時應迴避高空作業或地下工程等危險職業，日常多注意腳下安全。
