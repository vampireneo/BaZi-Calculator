# 天狗

## 定義

天狗，為凶神之一，性情剛烈暴躁，主意外傷害與血光之災。天狗取象天上惡星，主突發性禍事，尤與血光、手術、意外相關。

**類型**：凶神

## 含義

- 主是非口舌、爭鬥紛擾
- 易有意外傷害、血光之災
- 須防刀傷、車禍、手術
- 天狗性烈，主意外傷害，不可輕忽

## 起例方法

以**年支**查對應天狗地支，若該地支出現於其他柱位，即為天狗。

| 年支 | 天狗位 |
|------|------|
| 子   | 戌   |
| 丑   | 亥   |
| 寅   | 子   |
| 卯   | 丑   |
| 辰   | 寅   |
| 巳   | 卯   |
| 午   | 辰   |
| 未   | 巳   |
| 申   | 午   |
| 酉   | 未   |
| 戌   | 申   |
| 亥   | 酉   |

**注意**：天狗與弔客查法相同，但性質有別。弔客主喪事，天狗主血光。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 年柱 | 幼年易有意外傷害，祖輩有血光之象 |
| 月柱 | 青壯年易遭意外，工作環境有危險 |
| 日柱 | 本身或配偶易有血光，須注意日常安全 |
| 時柱 | 晚年或子女有意外之虞，出行需謹慎 |

## 傳統解說

《星平會海》載：「天狗星動，血光難免，須防刀兵水火之厄。」天狗為天界惡星，性如烈犬，見之則兇象環生。命帶天狗者，性格多較急躁，易與人衝突，引發是非。逢天狗流年，宜戒急用忍，避免從事高危活動，不宜進刀房手術（非必要情況），駕車需格外謹慎。若命局中有吉神制化，凶象可減輕。宜多行善積德，化解天狗煞氣。
