# 天哭

## 定義

天哭為凶星之一，主悲傷哭泣之事，多應於情感或親人方面的憂患，命帶此星者易有悲情遭遇。

## 性質

- **類型**：凶星
- **取法**：依年支起算

## 計算方式

以出生年支查對應地支，該地支所在柱即為天哭所在位置。

| 年支 | 天哭落支 |
|------|----------|
| 子   | 戌       |
| 丑   | 亥       |
| 寅   | 子       |
| 卯   | 丑       |
| 辰   | 寅       |
| 巳   | 卯       |
| 午   | 辰       |
| 未   | 巳       |
| 申   | 午       |
| 酉   | 未       |
| 戌   | 申       |
| 亥   | 酉       |

## 各柱影響

| 所在柱 | 影響                           |
|--------|-------------------------------|
| 年柱   | 幼年多悲苦，祖緣薄弱           |
| 月柱   | 中年多憂愁，工作壓力大         |
| 日柱   | 感情多悲傷，婚姻易有波折       |
| 時柱   | 子女緣薄，晚年多孤寂之感       |

## 傳統解說

天哭主悲傷哭泣之事，多與情感離別、親人喪亡相關。命帶天哭者，一生中易遭憂患，情感上多傷心之事。宜培養樂觀心態，廣結善緣，可緩和其凶性。與喪門、弔客同柱，則喪事之應驗更強。
