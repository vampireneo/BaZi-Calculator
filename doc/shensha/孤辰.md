# 孤辰

## 定義

孤辰屬**凶煞**，為八字神煞之一。主孤獨寂寞、男命克妻、宜晚婚。孤辰為男性孤獨之星，命中帶此星者，感情路上多波折，與伴侶聚少離多。

## 起例方法

以**年支**取之，查非年柱的其餘柱位（月、日、時）是否帶有孤辰之支。

| 年支所在季節組 | 孤辰所在支 |
|:---:|:---:|
| 亥子丑（冬） | 寅 |
| 寅卯辰（春） | 巳 |
| 巳午未（夏） | 申 |
| 申酉戌（秋） | 亥 |

**查法**：先以年支確定所屬季節組，再對照上表找出孤辰之支。若月柱、日柱或時柱的地支出現孤辰之支，即為命帶孤辰。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 月柱 | 青壯年感情孤立，兄弟緣薄 |
| 日柱 | 婚姻感情最受影響，配偶緣分薄 |
| 時柱 | 晚年孤獨，子女緣薄或離鄉 |

## 傳統解釋

孤辰與寡宿合稱「孤寡」，是八字中論婚姻與孤獨最常用的神煞之一。孤辰為前方空缺之意，陽氣孤行無伴，故男命帶此星者，感情上易陷入孤立狀態。

古書云：「孤辰寡宿最難當，四柱逢之主孤傷。」男命帶孤辰，感情宜晚婚，並宜注重溝通，以化解孤離之象。若孤辰與吉星同柱，孤獨之氣可稍得緩解。
