# 闌干

## 定義

闌干為八字神煞之一，屬**凶煞**。闌干如欄杆阻隔，主前進受阻，命帶此煞者諸事多障礙，難以突破重圍，前進之路常遭攔截。

## 含義

- 主阻隔不通、事多障礙、難有突破
- 凡事易遇無形阻力，進展緩慢
- 人際關係易受阻礙，溝通不暢

## 起例方式

依**年支**查對應地支，若該地支出現在命盤四柱中，即為帶闌干。

### 年支與闌干對照表

| 年支 | 闌干地支 |
|------|---------|
| 子   | 卯 |
| 丑   | 辰 |
| 寅   | 巳 |
| 卯   | 午 |
| 辰   | 未 |
| 巳   | 申 |
| 午   | 酉 |
| 未   | 戌 |
| 申   | 亥 |
| 酉   | 子 |
| 戌   | 丑 |
| 亥   | 寅 |

### 計算步驟

1. 取命盤年柱地支（年支）
2. 查上表得闌干對應地支
3. 若月柱、日柱或時柱地支中有對應地支，即帶闌干

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 甲   | **子**（查表：闌干為卯）|
| 月柱 | 丙   | 午   |
| 日柱 | 戊   | **卯**（命中闌干）|
| 時柱 | 庚   | 申   |

年支「子」對應闌干地支「卯」，日柱地支為「卯」，故帶闌干。

## 各柱闌干效應

| 闌干所在柱位 | 影響範疇 |
|-------------|---------|
| 月柱 | 事業多阻，升遷困難 |
| 日柱 | 婚姻感情受阻，配偶緣薄 |
| 時柱 | 子女緣薄，晚年多阻 |

## 傳統解說

闌干之象，猶如道路上設有欄杆，行人無法自由通行，必須繞道而行或尋求通融。命帶闌干者，凡事宜迂迴進取，切忌硬碰硬，應善用柔和手段化解阻礙，方能逐步突破困局，達成目標。
