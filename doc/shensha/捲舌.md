# 捲舌

## 定義

捲舌為八字神煞之一，屬**凶煞**。捲舌主口舌之災，命帶此煞者說話不當易招惹是非，口舌糾紛頻繁，宜謹言慎行，三思而後言。

## 含義

- 主口舌是非、言語不慎、易惹爭端
- 說話易得罪人，引發不必要的衝突
- 與人爭辯是非，難以善終

## 起例方式

依**年支**查對應地支，若該地支出現在命盤四柱中，即為帶捲舌。

### 年支與捲舌對照表

| 年支 | 捲舌地支 |
|------|---------|
| 子   | 戌 |
| 丑   | 亥 |
| 寅   | 子 |
| 卯   | 丑 |
| 辰   | 寅 |
| 巳   | 卯 |
| 午   | 辰 |
| 未   | 巳 |
| 申   | 午 |
| 酉   | 未 |
| 戌   | 申 |
| 亥   | 酉 |

### 計算步驟

1. 取命盤年柱地支（年支）
2. 查上表得捲舌對應地支
3. 若月柱、日柱或時柱地支中有對應地支，即帶捲舌

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 庚   | **子**（查表：捲舌為戌）|
| 月柱 | 壬   | 午   |
| 日柱 | 甲   | **戌**（命中捲舌）|
| 時柱 | 丙   | 寅   |

年支「子」對應捲舌地支「戌」，日柱地支為「戌」，故帶捲舌。

## 各柱捲舌效應

| 捲舌所在柱位 | 影響範疇 |
|-------------|---------|
| 年柱 | 幼年口舌多，家庭爭吵頻繁 |
| 月柱 | 職場口舌是非多，與同事易起衝突 |
| 日柱 | 夫妻口角不斷，婚姻關係緊張 |
| 時柱 | 晚年口舌多，與子女易有言語衝突 |

## 傳統解說

捲舌之象，猶如舌頭捲曲糾纏，說出的話難以圓滑得體，反而惹禍上身。命帶捲舌者，說話前宜三思，避免在公開場合發表敏感言論，尤宜遠離搬弄是非之人。修煉沉默是金之道，以柔和的溝通方式替代爭辯，方能化解口舌之災。
