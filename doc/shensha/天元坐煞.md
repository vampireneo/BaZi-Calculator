# 天元坐煞

## 定義

天元坐煞（Tiān Yuán Zuò Shā）為八字神煞之一，屬**凶煞**。天元即日柱天干（日主），坐煞表示日主坐下之地支對天干有克制或沖剋之力，主性格急躁、易有衝突、須防意外。

## 含義

- 主性格急躁、易衝動行事
- 易與人產生衝突或口角
- 須防意外傷害、血光之災
- 人際關係較為緊張，易得罪他人

## 起法（以日柱干支查）

天元坐煞以**特定日柱干支組合**論之，凡日干受日支所克或相沖者即是：

| 日柱組合 | 干支關係 |
|---------|---------|
| 甲申 | 木被金克 |
| 乙酉 | 木被金克 |
| 丙子 | 火被水克 |
| 丁亥 | 火被水克 |
| 戊寅 | 土被木克 |
| 己卯 | 土被木克 |
| 庚午 | 金被火克 |
| 辛巳 | 金被火克 |
| 壬申 | 水坐申（金生水，然申含庚金七殺）|
| 癸酉 | 水坐酉（金生水，然酉含辛金七殺）|

**查法**：對照自身日柱干支，若符合上表任一組合，即命帶天元坐煞。

## 各柱影響

| 情況 | 影響 |
|------|------|
| 日柱帶此煞 | 日主受剋，本身壓力大，易急躁衝動 |
| 命局有印化 | 印星洩煞生日元，可緩和衝突之象 |
| 行運逢煞旺 | 衝突、意外之事更易發生，宜謹慎 |

## 傳統解說

天元坐煞表示日主坐下受剋，如人站立於險地，隨時面臨壓力與威脅。古籍云：「天元坐煞，坐立不安，凡事多阻，性急難成。」命帶此煞者，自身承受的壓力往往較大，處事易操之過急，因急躁而引發不必要的衝突或意外。宜修身養性，凡事三思而後行，以柔克剛，方能化解坐下之煞氣。
