# 寡宿

## 定義

寡宿屬**凶煞**，為八字神煞之一。主孤獨寂寞、女命克夫、宜晚婚。寡宿為女性孤獨之星，命中帶此星者，婚姻感情多有缺憾，與伴侶易生隔閡或分離。

## 起例方法

以**年支**取之，查非年柱的其餘柱位（月、日、時）是否帶有寡宿之支。

| 年支所在季節組 | 寡宿所在支 |
|:---:|:---:|
| 亥子丑（冬） | 戌 |
| 寅卯辰（春） | 丑 |
| 巳午未（夏） | 辰 |
| 申酉戌（秋） | 未 |

**查法**：先以年支確定所屬季節組，再對照上表找出寡宿之支。若月柱、日柱或時柱的地支出現寡宿之支，即為命帶寡宿。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 月柱 | 青壯年感情受阻，姐妹緣薄 |
| 日柱 | 婚姻感情最受影響，夫妻易生離 |
| 時柱 | 晚年孤守，子女緣薄或天各一方 |

## 傳統解釋

寡宿與孤辰合稱「孤寡」，孤辰論男之孤，寡宿論女之寡。寡宿為後方殘缺之意，陰氣孤守無依，故女命帶此星者，婚姻中易扮演付出多、收穫少的一方。

古書云：「孤辰寡宿最難當，四柱逢之主孤傷。」女命帶寡宿，感情宜晚婚，重視自身修養與獨立，以穩固婚姻關係。若命格旺盛且吉星輔助，可從事宗教或社會服務，以孤化貴。
