# 小耗

## 定義

小耗為歲破之前一位地支，即十二地支循環中緊接歲破之前一支。小耗主輕微財物耗損、暗中耗散，其損傷程度較大耗為輕，多為不知不覺中財物流失，為命理中財耗類凶煞之一。

## 性質

**類型**：凶星

**主要含義**：主小破財、暗中耗損、宜節儉。小耗損傷較輕，多為暗中耗散，往往不易察覺，積少成多亦可造成相當損失。

## 起例方法

以**年支**先查歲破，再取歲破之**前一位**地支（十二支逆序循環），即為小耗。

| 年支 | 歲破 | 小耗（歲破前一位） |
|-----|------|-----------------|
| 子  | 午   | 巳              |
| 丑  | 未   | 午              |
| 寅  | 申   | 未              |
| 卯  | 酉   | 申              |
| 辰  | 戌   | 酉              |
| 巳  | 亥   | 戌              |
| 午  | 子   | 亥              |
| 未  | 丑   | 子              |
| 申  | 寅   | 丑              |
| 酉  | 卯   | 寅              |
| 戌  | 辰   | 卯              |
| 亥  | 巳   | 辰              |

> **推算說明**：十二地支順序為子→丑→寅→卯→辰→巳→午→未→申→酉→戌→亥，歲破前一位即逆行一步所得之支。

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 祖業有暗中耗損，早年需節儉 |
| 月柱   | 事業收入易有漏財，中年財運不穩 |
| 日柱   | 個人財物暗耗，生活需精打細算 |
| 時柱   | 晚年小有破財，或因子女花費較多 |

## 傳統解說

小耗為財物暗損之象，傳統命理認為小耗雖損傷較大耗為輕，但其特點在於「暗」字——財物在不知不覺中流失，難以察覺其去向。命中逢小耗，宜養成節儉習慣，做好財務記錄，避免因疏忽而導致財物悄然耗散。流年行運逢小耗，亦宜謹慎理財。
