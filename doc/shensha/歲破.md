# 歲破

## 定義

歲破為流年太歲所衝之地支，與太歲六沖相對，故名「歲破」。歲破所在之月稱「破月」，凡太歲與之相沖，主諸事破敗、財物損耗，為命理中重要之凶煞。

## 性質

**類型**：凶星

**主要含義**：主破耗損財、諸事不順、宜守不宜攻。逢歲破之年或命中帶歲破，諸事多有阻滯，不宜輕舉妄動，以守為上策。

## 起例方法

以**年支**查對應地支（即年支之六沖），凡四柱（月、日、時）地支中出現對應地支，即為命中帶歲破。

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 月柱   | 家業難守，中年事業多波折 |
| 日柱   | 婚姻感情易生破裂，身體較差 |
| 時柱   | 晚景不豐，子息緣薄 |

## 傳統解說

歲破與太歲相沖，沖則破，故名歲破。傳統命理認為，凡命局逢歲破，如動土、嫁娶、開業諸事，皆不宜進行，以免破財損業。流年行運若再逢歲破，則破耗尤重，宜靜守待時，切忌冒進。
