# 亡神

## 定義

亡神屬**凶煞**，為八字神煞之一。主心神不寧、易有是非口舌、須防暗耗。命中帶亡神者，情緒較不穩定，易受外界干擾，財物亦有無形耗散之象。

## 起例方法

以**日支**或**年支**所在三合局的後一位（亡位）取之。

| 日支／年支所在三合 | 亡神所在支 |
|:---:|:---:|
| 申子辰（水局） | 亥 |
| 寅午戌（火局） | 巳 |
| 巳酉丑（金局） | 申 |
| 亥卯未（木局） | 寅 |

**查法**：以日支（或年支）確定所屬三合局，再對照上表找出對應地支。若該地支出現在命盤的年、月、日、時任一柱，即為帶亡神。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 幼年家境有暗耗，長輩緣薄 |
| 月柱 | 工作中易有是非，財務不穩 |
| 日柱 | 心神易浮動，婚姻或有口舌 |
| 時柱 | 晚年思慮過多，子女關係有隔閡 |

## 傳統解釋

亡神古稱「亡殺」，與劫煞同出三合局，劫煞取前，亡神取後，兩者常並論。亡神象徵陽氣消散、精神渙散之意，故主心神不寧、暗耗無形。

命帶亡神者，宜修身養性，避免與人口角；理財宜保守，慎防詐騙與無謂消費。若亡神與劫煞同時出現，稱「劫煞亡神俱全」，須格外謹慎人身財務安全。
