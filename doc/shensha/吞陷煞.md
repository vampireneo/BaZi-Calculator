# 吞陷煞

## 定義

吞陷煞為八字神煞之一，屬**凶煞**。「吞陷」意指陷入困境如入泥沼，難以自拔，主命主易遭困厄、事業受阻、處境艱難。

## 含義

- 主陷入困境、難以自拔
- 事業易遭挫折，謀事多阻
- 財運不穩，易有破財之象
- 宜謹慎行事，切忌冒進

## 起算方式

以**日干**查吞陷煞所在之地支，若該地支出現於命盤其他柱位，即為吞陷煞。

### 對照表

| 日干         | 吞陷煞位置 |
|--------------|------------|
| 甲、乙（木） | 辰         |
| 丙、丁（火） | 戌         |
| 戊、己（土） | 辰         |
| 庚、辛（金） | 戌         |
| 壬、癸（水） | 辰         |

> 規律：木、土、水日干對應辰；火、金日干對應戌。辰為水庫，戌為火庫，皆為蓄藏沉陷之地。

## 各柱位影響

| 柱位 | 影響說明 |
|------|----------|
| 年柱 | 祖業難守，早年環境困頓 |
| 月柱 | 事業起伏大，中年多有挫折 |
| 日柱 | 個人易陷困境，婚姻宮有阻 |
| 時柱 | 晚年有困厄，子女緣薄 |

## 傳統解說

吞陷煞古謂人陷入如泥沼般之困境，一旦深陷其中，便難以脫身。命帶此煞者，凡事宜謹慎規劃，切勿輕率冒進，否則極易一步步深陷麻煩之中。辰為濕土，戌為燥土，皆有蓄藏、深陷之義，故此煞取象於此。遇此煞宜保守穩健，以守代攻，量力而為，方可化解困境。
