# 祿神

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 吉神 |
| 別名 | 祿星、正祿 |
| 取用基準 | 日干 |

## 定義與含義

祿神為命盤中象徵衣食財祿與福氣的重要吉神。命帶祿神者，主衣食無憂、財祿豐厚、福氣綿長，一生生活富足，飲食有保障，工作收入穩定，是命盤中最基本的福祿象徵之一。

## 起算方法

以**日干**為基準，查看四柱地支中是否出現對應地支。

| 日干 | 祿神地支 |
|------|---------|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙 | 巳 |
| 丁 | 午 |
| 戊 | 巳 |
| 己 | 午 |
| 庚 | 申 |
| 辛 | 酉 |
| 壬 | 亥 |
| 癸 | 子 |

## 判斷步驟

1. 找出日干（日柱天干）
2. 對照上表查出對應地支
3. 在年、月、日、時四柱地支中搜尋該地支
4. 出現即代表命帶祿神

## 各柱位置與特殊名稱

| 柱位 | 名稱 | 效應說明 |
|------|------|---------|
| 年柱 | 歲祿 | 祖上有福蔭，幼年生活無虞 |
| 月柱 | 建祿 | 一生財祿穩健，職場收入佳，最為有力 |
| 日柱 | 專祿 | 自身福祿深厚，配偶亦有財緣 |
| 時柱 | 歸祿 | 晚年生活富足，子女孝順，老來無憂 |

## 祿神特殊格局

| 格局 | 說明 |
|------|------|
| 建祿格 | 月支為祿，格局清正，適合靠自身能力致富 |
| 祿逢沖 | 祿被沖破，財祿受損，生計易有波動 |
| 祿逢合 | 祿被合去，財祿被他人分享或奪取 |
| 祿馬同鄉 | 祿與驛馬同柱，在奔波中得財，宜外出發展 |

## 傳統解說

祿在五行中代表日干的「臨官」之地，即日干五行之氣達到旺盛的地支。古人認為祿代表官祿、俸祿，是衣食俸祿的根本。

建祿格（月支為祿）是傳統命理中的重要格局之一，命主往往依靠自身才幹立業，不善依賴他人，自食其力，財祿穩定。此格最宜見官星，則可得官祿雙全之命。

祿神最喜透出天干、與正官同現，則財官雙美，一生富貴。最忌被沖被合，否則財祿不穩。

> **注意**：祿神（建祿）不同於印星，建祿格之人通常以自身努力取財，不善繼承祖業或依賴貴人。
