# 魁罡

## 定義

魁罡屬**中性煞**（亦稱「剛星」），為八字神煞中較為特殊者。主性格剛毅、有膽識魄力，但須防剛愎自用。命帶魁罡者，意志力堅強，處事果決，具有天生的領導氣場，然性格過剛易折，人際關係需加以磨合。

## 起例方法

魁罡以**日柱**（天干＋地支的完整組合）論之，非單取地支，須天干與地支皆符方成立。

| 魁罡日柱 | 五行說明 |
|:---:|:---|
| 庚辰 | 庚金坐辰，金剛土燥 |
| 庚戌 | 庚金坐戌，金猛火庫 |
| 壬辰 | 壬水坐辰，水盛龍宮 |
| 戊戌 | 戊土坐戌，雙土剛烈 |

**查法**：確認日柱的天干與地支組合，是否完整符合上表四組之一。僅地支相同而天干不符，不算魁罡。

## 各日柱的特質差異

| 魁罡日柱 | 性格傾向 |
|:---:|:---|
| 庚辰 | 思維縝密，執行力強 |
| 庚戌 | 性格最為剛烈，意志如鐵 |
| 壬辰 | 智謀兼備，氣勢磅礡 |
| 戊戌 | 固執保守，但誠信踏實 |

## 傳統解釋

魁罡古稱「天罡」，取北斗七星中魁星與罡星之意，象徵剛毅不屈的天地之氣聚於一柱。古書云：「魁罡最喜身旺，身強則英雄本色，身弱則剛愎招禍。」

命帶魁罡者，若命格強旺，可成就非凡事業，適合從事軍警、法律、外科、工程等需要果決與魄力的領域。若命格過弱或多逢沖剋，則易因過於固執而招致挫折，須學習柔韌處世之道。女命帶魁罡，傳統上認為個性較強，婚姻需雙方互相包容。
