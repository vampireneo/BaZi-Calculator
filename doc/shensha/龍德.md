# 龍德

## 定義

龍德（Lóng Dé）為八字神煞之一，屬**吉神**。龍德乃陰中之陽神，吉力強大，能逢凶化吉、遇難呈祥，為命局中最具庇護力量的吉星之一。

## 含義

- 主逢凶化吉、龍德庇佑、遇難呈祥
- 遇凶煞時能減輕其負面影響
- 增強命局的貴氣與吉祥之力

## 各柱影響

| 所在柱 | 影響 |
|--------|------|
| 年柱 | 祖輩庇蔭深厚，家族有貴氣 |
| 月柱 | 父母扶持有力，事業逢凶化吉 |
| 日柱 | 本身帶貴，婚姻和諧，凶事能化 |
| 時柱 | 晚年吉祥，子女有助，老來安泰 |

## 傳統解說

龍德為陰中之陽神，性質純吉。古籍云：「龍德加臨，百事吉祥，逢凶不凶，遇難自解。」命帶龍德者，一生中即便身陷困境，亦常有貴人相助、化險為夷之機緣。此星吉力甚強，尤以命局凶煞較重時，龍德之庇護作用更為明顯。
