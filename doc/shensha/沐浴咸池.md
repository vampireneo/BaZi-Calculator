# 沐浴咸池

## 定義

沐浴咸池是「沐浴」（十二長生之一）與「咸池」（桃花星）兩種神煞同時出現於同一地支位置的特殊組合。當日干所對應的沐浴位，與年支或日支所對應的桃花位（咸池）重合於命局中，即稱為沐浴咸池。

**類型**：中
**主要含義**：主桃花旺盛、異性緣佳、須防感情糾紛。沐浴咸池代表桃花與沐浴同位，感情方面特別旺盛。

---

## 計算方法

**第一步**：依日干確認沐浴位：

| 日干 | 沐浴位 |
|------|--------|
| 甲 | 子 |
| 乙 | 巳 |
| 丙、戊 | 卯 |
| 丁、己 | 申 |
| 庚 | 午 |
| 辛 | 亥 |
| 壬 | 酉 |
| 癸 | 寅 |

**第二步**：依年支或日支的三合局確認咸池（桃花）位：

| 地支 | 三合局 | 咸池位 |
|------|--------|--------|
| 申、子、辰 | 水局 | 酉 |
| 寅、午、戌 | 火局 | 卯 |
| 巳、酉、丑 | 金局 | 午 |
| 亥、卯、未 | 木局 | 子 |

**第三步**：若沐浴位＝咸池位，且此地支出現於命局中，即構成沐浴咸池。

---

## 傳統解說

沐浴在十二長生中代表「沐浴更衣、展現魅力」之意，本身即帶有吸引異性的特質；咸池（桃花）則是異性緣旺盛的標誌。兩者重疊於同一地支，感情磁場加倍旺盛，異性緣極佳，然而過旺之桃花往往帶來感情困擾或糾紛。命帶沐浴咸池者，外表吸引力強，人緣極好，但需謹慎處理感情，避免多角關係或感情糾葛，方能將旺盛的異性緣轉化為正面的人際資源。
