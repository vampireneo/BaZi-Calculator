# 血刃

## 定義

血刃又稱「血支」，依日干查對應地支，為主刀傷血光的凶煞。

## 類型

**凶煞**

## 含義

主血光之災、須防刀傷手術、謹慎行事。血刃如利刃見血，象徵利器傷人，命中帶此煞者，易遭刀傷、意外割傷或需手術治療，日常生活中應避免接觸刀具利器。

## 起例方法

依**日干**查對應地支，若該地支出現於四柱中即帶血刃：

| 日干 | 血刃地支 |
|------|---------|
| 甲   | 卯       |
| 乙   | 辰       |
| 丙   | 午       |
| 丁   | 未       |
| 戊   | 午       |
| 己   | 未       |
| 庚   | 酉       |
| 辛   | 戌       |
| 壬   | 子       |
| 癸   | 丑       |

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 幼年曾有刀傷，或體質易出血 |
| 月柱 | 職業相關刀傷風險，廚師、外科醫生需注意 |
| 日柱 | 本人血刃最旺，須防各類刀傷 |
| 時柱 | 晚年或子女有血光之憂 |

## 傳統解釋

血刃在古命書中是重要的血光凶煞，與流霞煞性質相近，但血刃更強調刀器所傷。凡帶血刃者，古人建議從事醫療、廚藝等以刀為業之職，可借職業之刀化解私人之刀傷，以主動化被動。平日烹飪、剪切時須加倍小心，避免無謂的刀傷意外發生。
