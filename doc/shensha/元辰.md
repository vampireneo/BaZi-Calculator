# 元辰

## 定義

元辰，又稱「元辰煞」，以年支及性別為基準起算。元辰主耗散破敗之事，為財帛宮之凶神，象徵錢財難聚、諸事耗損。

**類型**：凶神

## 含義

- 主耗散破敗、財物難積
- 諸事不順、意外損失
- 須謹慎理財、避免浪費
- 易有破財之象，錢財如流水

## 起例方法

以**年支**配合性別查對應元辰地支，若該地支出現於其他柱位，即為元辰。

### 陽年支（子、寅、辰、午、申、戌）

| 年支 | 元辰位 |
|------|------|
| 子   | 未   |
| 寅   | 酉   |
| 辰   | 亥   |
| 午   | 丑   |
| 申   | 卯   |
| 戌   | 巳   |

### 陰年支（丑、卯、巳、未、酉、亥）

| 年支 | 元辰位 |
|------|------|
| 丑   | 午   |
| 卯   | 申   |
| 巳   | 戌   |
| 未   | 子   |
| 酉   | 寅   |
| 亥   | 辰   |

**例**：年支為子（陽支），元辰位為未。若月、日、時柱地支有未，即為元辰。

## 各柱位置效應

| 位置 | 效應說明 |
|------|---------|
| 年柱 | 祖業耗散，早年財運不佳，家境易中落 |
| 月柱 | 事業財運多耗損，工作收入不穩，易破財 |
| 日柱 | 本身財運受損，夫妻宮見財耗，婚姻有財務糾紛 |
| 時柱 | 晚年財運有損，子女花費大，晚景需量入為出 |

## 傳統解說

《星平會海》云：「元辰者，耗星也，所到之處，財物難留。」元辰之性如漏斗，財進財出，難以蓄積。命帶元辰者，一生中易有意外破財之事，或因人際應酬、感情債務而耗財。宜從年輕時建立儲蓄習慣，量力而行，不宜過度借貸或為他人作保。若命局財星有力，則元辰之破財力量稍減，但仍需謹慎理財，方能守住財富。
