# 五鬼

## 定義

五鬼，又稱五虛，為主陰邪小人之凶煞。「五鬼」源自道教中五方鬼神之說，在命理中代表陰暗、詭秘、小人暗算之象。凡命中帶五鬼，須防小人陷害及暗中之是非。

## 性質

**類型**：凶星

**主要含義**：主小人暗害、是非口舌、須防陰謀詭計。五鬼又稱五虛，主陰邪之事，凡逢此煞，身邊易有小人，宜提防背後中傷。

## 起例方法

以**年支**查對應地支，凡四柱（月、日、時）地支中出現對應地支，即為命中帶五鬼。

| 年支 | 五鬼地支 |
|-----|---------|
| 子  | 辰      |
| 丑  | 巳      |
| 寅  | 午      |
| 卯  | 未      |
| 辰  | 申      |
| 巳  | 酉      |
| 午  | 戌      |
| 未  | 亥      |
| 申  | 子      |
| 酉  | 丑      |
| 戌  | 寅      |
| 亥  | 卯      |

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 幼年家中易有陰邪之事，或祖上有小人之患 |
| 月柱   | 工作環境中易有小人，職場人際關係複雜 |
| 日柱   | 本人易遭暗算，夫妻之間易有第三者介入 |
| 時柱   | 晚年須防小人，或子女易惹是非 |

## 傳統解說

五鬼源自古代道教「五方五鬼」之觀念，入命主陰邪暗算之事。傳統命理認為，命中逢五鬼，易招小人嫉妒與陷害，常有背後搬弄是非之人。此煞較官符更為陰暗，官符尚可對簿公堂，五鬼則多為暗中為害，難以防範。命逢五鬼，宜廣結善緣，行事低調，避免樹敵，同時注意家中風水，以化解陰煞之氣。
