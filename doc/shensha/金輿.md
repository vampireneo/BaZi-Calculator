# 金輿

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 吉神 |
| 別名 | 金輿祿、金車 |
| 取用基準 | 日干 |

## 定義與含義

金輿為命盤中象徵出行平安與交通福祿的吉神。命帶金輿者，主出行平安、有車馬之福、利於交通出行，古代象徵乘坐華麗馬車出行的貴族，現代則引申為利於駕車、出差、旅行，鮮少遭遇交通事故，出行順利。

## 起算方法

以**日干**為基準，查看四柱地支中是否出現對應地支。

| 日干 | 金輿地支 |
|------|---------|
| 甲 | 辰 |
| 乙 | 巳 |
| 丙 | 未 |
| 丁 | 申 |
| 戊 | 未 |
| 己 | 申 |
| 庚 | 戌 |
| 辛 | 亥 |
| 壬 | 丑 |
| 癸 | 寅 |

## 判斷步驟

1. 找出日干（日柱天干）
2. 對照上表查出對應地支
3. 在年、月、日、時四柱地支中搜尋該地支
4. 出現即代表命帶金輿

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 祖上有車馬之福，家族出行順利，幼年少交通意外 |
| 月柱 | 工作中出行平安，商旅順遂，公務出差少災 |
| 日柱 | 個人出行有保護，喜愛旅遊或移動性工作 |
| 時柱 | 晚年出行平安，子女有交通福分 |

## 金輿與相關神煞的關係

| 比較神煞 | 關係說明 |
|---------|---------|
| 驛馬 | 驛馬主奔波移動，金輿主出行平安，兩者同現則移動中有保護 |
| 文昌 | 金輿與文昌同現，出行中亦有學習機會，利於遊學 |
| 天乙貴人 | 金輿與天乙同現，出行貴人相助，逢難有救 |

## 金輿特殊格局

| 格局 | 說明 |
|------|------|
| 金輿帶馬 | 金輿與驛馬同現，一生出行頻繁且平安，適合旅遊業或運輸業 |
| 金輿逢沖 | 出行保護減弱，交通意外風險上升，宜謹慎駕車 |
| 金輿臨官 | 出行中得官位，適合因公出差或外派任職 |
| 金輿帶財 | 出行中得財，適合業務跑外勤或旅遊相關行業 |

## 傳統解說

金輿古稱「金車」，即以金飾裝點的豪華馬車，是古代貴族出行的交通工具。命帶金輿者，如同搭乘金車出行，旅途順遂，鮮少遭遇意外。

古代命理重視出行安危，金輿被視為保護旅途平安的吉神。凡商旅、征戰、官差等需要長途跋涉者，命帶金輿則大吉。

在現代社會，金輿的應用範圍擴展至各類交通出行，包括開車、搭乘大眾運輸、搭飛機旅行等。命帶金輿之人，在道路安全方面較有神煞保護，但並非代表絕對無虞，仍需謹守交通安全規則。

金輿亦有象徵財富的含義，古代金車代表富貴，故金輿入命亦有財富豐厚、生活奢華之象，與財星同現時此效果更為明顯。

> **注意**：金輿最忌被沖破，若金輿所在地支受沖，出行保護力減弱，宜特別注意交通安全。
