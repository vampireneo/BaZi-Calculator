# 六秀日

## 定義

六秀日為八字神煞之一，屬**吉神**，主聰明秀氣、才華出眾、氣質非凡。六秀日生人多才多藝，氣質出眾，廣受人愛戴。

## 含義

- 聰明秀氣，才思敏捷
- 才華出眾，多才多藝
- 氣質非凡，外形出眾
- 文學藝術方面有天賦
- 廣受他人喜愛與欣賞

## 起法（以日柱干支判斷）

六秀日以**日柱干支**為準，以下六組干支日柱即帶此神：

| 序號 | 日柱 | 特性說明 |
|------|------|----------|
| 1    | 丙午 | 丙火午火，純陽之火，光明磊落 |
| 2    | 丁未 | 丁火未土，柔中帶秀，溫文爾雅 |
| 3    | 戊子 | 戊土子水，剛柔並濟，才智兼備 |
| 4    | 己丑 | 己土丑土，厚德載物，才德兼修 |
| 5    | 癸巳 | 癸水巳火，水火交融，聰慧靈動 |
| 6    | 癸酉 | 癸水酉金，金水相生，智慧超群 |

> 凡日柱干支屬以上六組之一，即命帶六秀，主聰明秀氣。

## 各柱影響

六秀日以日柱干支論斷，主要影響本人之才華與氣質：

| 情況 | 影響 |
|------|------|
| 命帶六秀 | 天生聰慧，氣質出眾，才藝過人 |
| 六秀逢文昌 | 才華更為突出，學業優異，易成文名 |
| 六秀逢財官 | 才華得以施展，事業財運皆豐 |
| 六秀逢凶煞 | 雖有才華，但恃才傲物，需注意謙遜 |

## 適合發展方向

命帶六秀日之人，尤其適合以下領域：

- 文學、詩詞、寫作
- 藝術、音樂、表演
- 教育、學術研究
- 設計、美學相關行業

## 傳統解說

古籍云：「六秀日生，才氣過人。」六秀日之得名，乃因此六日天干地支之配合，五行之氣最為清秀靈動，能激發人之聰明才智與藝術氣質。命帶六秀日者，往往給人留下深刻印象，才華自然流露，不需刻意表現便能脫穎而出，為命理中論才華最重要之吉神之一。
