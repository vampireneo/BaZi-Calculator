# 劫煞

## 定義

劫煞屬**凶煞**，為八字神煞之一。主易有劫難、須防小人暗害、謹慎理財。命中帶劫煞者，一生中較易遭遇財物損失或人身安全方面的威脅，須提高警覺。

## 起例方法

以**日支**或**年支**所在三合局的前一位（劫位）取之。

| 日支／年支所在三合 | 劫煞所在支 |
|:---:|:---:|
| 申子辰（水局） | 巳 |
| 寅午戌（火局） | 亥 |
| 巳酉丑（金局） | 寅 |
| 亥卯未（木局） | 申 |

**查法**：以日支（或年支）確定所屬三合局，再對照上表找出對應地支。若該地支出現在命盤的年、月、日、時任一柱，即為帶劫煞。

## 各柱位置的影響

| 柱位 | 影響 |
|:---:|:---|
| 年柱 | 幼年易有意外，祖業難守 |
| 月柱 | 事業易有波折，須防職場小人 |
| 日柱 | 本身易遭劫難，財務須謹慎 |
| 時柱 | 晚年或子女有劫難之憂 |

## 傳統解釋

劫煞古稱「劫殺」，取三合局中被劫奪之意。三合局代表完整之氣，劫煞之支正是截斷此氣的關鍵位置，象徵外來力量的突然侵害。

命帶劫煞者，宜謹慎保管財物，避免與人合夥糾紛，並注意出行安全。若與吉星同現，可化解部分凶意；若再逢刑沖，凶象加重，須格外留意。
