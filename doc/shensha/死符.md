# 死符

## 定義

死符為主疾厄病患之凶煞，「死符」取義於死亡之符兆，在命理中代表健康衰弱、疾病纏身之象。凡命中帶死符，須特別注意身體健康，防患於未然。

## 性質

**類型**：凶星

**主要含義**：主疾病纏身、健康不佳、須注意保養。死符主疾厄，須防病患纏身，凡逢此煞，易有慢性疾病或身體虛弱之象。

## 起例方法

以**年支**查對應地支，凡四柱（月、日、時）地支中出現對應地支，即為命中帶死符。

| 年支 | 死符地支 |
|-----|---------|
| 子  | 巳      |
| 丑  | 午      |
| 寅  | 未      |
| 卯  | 申      |
| 辰  | 酉      |
| 巳  | 戌      |
| 午  | 亥      |
| 未  | 子      |
| 申  | 丑      |
| 酉  | 寅      |
| 戌  | 卯      |
| 亥  | 辰      |

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 幼年體弱，或幼年曾有重病 |
| 月柱   | 青壯年期間易有慢性病，須注重保健 |
| 日柱   | 本人體質較弱，易有疾厄，配偶健康亦需關注 |
| 時柱   | 晚年健康欠佳，或子女體質較弱 |

## 傳統解說

死符在傳統年神煞中屬主疾厄之凶神，與病符相類。傳統命理認為，命中逢死符，表示該流年或命局中健康運勢較差，易有疾病纏身之虞。死符入命，宜定期體檢，注意飲食起居，加強養生保健。若命局中死符與病符同現，則疾厄之象更為嚴重，務必重視身體健康，及早預防勝於治療。
