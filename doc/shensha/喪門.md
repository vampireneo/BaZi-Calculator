# 喪門

## 定義

喪門為主喪葬之事之凶煞，與弔客星相對而論。凡命中或流年帶喪門，主家宅中易有喪事發生，或與喪葬相關之憂患，須防孝服纏身。

## 性質

**類型**：凶星

**主要含義**：主喪服之事、須防孝服、家宅不寧。喪門主喪事，與弔客同論，凡逢此星，宜注意家中長輩健康，避免參與喜慶之事。

## 起例方法

以**年支**查對應地支，凡四柱（月、日、時）地支中出現對應地支，即為命中帶喪門。

| 年支 | 喪門地支 |
|-----|---------|
| 子  | 寅      |
| 丑  | 卯      |
| 寅  | 辰      |
| 卯  | 巳      |
| 辰  | 午      |
| 巳  | 未      |
| 午  | 申      |
| 未  | 酉      |
| 申  | 戌      |
| 酉  | 亥      |
| 戌  | 子      |
| 亥  | 丑      |

## 各柱影響

| 所在柱位 | 影響說明 |
|---------|---------|
| 年柱   | 祖上或幼年曾有喪事，家宅不寧 |
| 月柱   | 父母或兄弟姐妹易有喪憂 |
| 日柱   | 本人易逢喪事，婚姻配偶需注意健康 |
| 時柱   | 晚年或子女易有喪憂之事 |

## 傳統解說

喪門在傳統命理中屬太歲旁側之凶神，主管喪葬之事宜。古籍云：「喪門弔客，主孝服之災。」凡命局或流年逢喪門，應多關注家中長輩及親屬之身體健康，逢喪門之年不宜參加婚慶喜宴，亦不宜喬遷、動土等重大事務。若喪門與弔客同現，則喪事之兆更為明顯。
