# 羊刃

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 凶煞 |
| 別名 | 陽刃、劫煞 |
| 取用基準 | 日干（僅限陽干） |

## 定義與含義

羊刃為命盤中最具代表性的凶煞之一，象徵剛烈衝動與血光之災。命帶羊刃者，主性格剛烈、易有血光之災、須防意外傷害，個性強硬，不服輸，遇事容易衝動，缺乏圓融，易與人發生衝突。

## 起算方法

羊刃**僅陽干（陽日主）才有**，陰干無羊刃。以**日干**為基準，查看四柱地支中是否出現對應地支。

| 日干 | 羊刃地支 | 備註 |
|------|---------|------|
| 甲 | 卯 | 陽木 |
| 丙 | 午 | 陽火 |
| 戊 | 午 | 陽土 |
| 庚 | 酉 | 陽金 |
| 壬 | 子 | 陽水 |
| 乙、丁、己、辛、癸 | 無 | 陰干無羊刃 |

## 飛刃（對宮）

飛刃為羊刃地支的對沖地支，力量較羊刃稍弱，但亦具凶性。

| 日干 | 羊刃 | 飛刃（對沖） |
|------|------|------------|
| 甲 | 卯 | 酉 |
| 丙 | 午 | 子 |
| 戊 | 午 | 子 |
| 庚 | 酉 | 卯 |
| 壬 | 子 | 午 |

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 幼年多災，或家族中有剛烈之人 |
| 月柱 | 性格剛強，工作衝勁足，但易與上司衝突 |
| 日柱 | 個性剛烈不讓人，婚姻易有摩擦，配偶強勢 |
| 時柱 | 晚年身體易有傷，子女個性強硬 |

## 羊刃特殊格局

| 格局 | 說明 |
|------|------|
| 羊刃格 | 月支為羊刃，格局需配合官殺制化，方能成大器 |
| 羊刃駕殺 | 羊刃與七殺同現，剛強之氣極重，宜從武職或高壓行業 |
| 羊刃逢合 | 羊刃被合化，凶性減弱，性格趨於圓融 |
| 羊刃逢沖 | 凶性爆發，血光意外風險大增 |

## 傳統解說

羊刃又稱「陽刃」，其命名源於陽干五行之氣過旺、如同鋒利刀刃，難以控制。古人視羊刃為極旺之氣，過旺則傷身、傷財、傷官，是命盤中最需要制化的凶煞之一。

然而，羊刃並非全然為害。羊刃入命者，體魄強健，意志堅定，衝勁十足。若命盤中有七殺或正官制化羊刃，則剛強之氣得以收斂，可成就一番大事業，尤其適合軍警、外科、工程、競技等需要膽識與體力的行業。

歷史上，許多將帥政治強人的命盤中，常見羊刃格局。

> **注意**：羊刃喜見官殺制化，忌無制或逢沖，否則凶性難以控制。陰干（乙、丁、己、辛、癸）無羊刃，切勿誤算。
