# 災煞

## 定義

災煞又稱「災殺」，依日支或年支所屬三合局起算，為常見凶煞之一。

## 類型

**凶煞**

## 含義

主災禍臨身、須防水火之災、謹慎行事。命中帶災煞者，容易遭遇意外災難，尤其須注意水災、火災等天災人禍，行事宜加倍小心。

## 起例方法

依**日支**或**年支**所屬三合局查表：

| 日支／年支（三合局） | 災煞所在支 |
|----------------------|-----------|
| 申、子、辰（水局）   | 午         |
| 寅、午、戌（火局）   | 子         |
| 巳、酉、丑（金局）   | 卯         |
| 亥、卯、未（木局）   | 酉         |

> 查法：以日支（或年支）找到所屬三合局，再對應右欄地支，若該地支出現於四柱中，即為帶災煞。

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 早年易逢災禍，祖業受損 |
| 月柱 | 中年事業波折，防職業傷害 |
| 日柱 | 本身易遭意外，健康需注意 |
| 時柱 | 晚年多事，子女防災 |

## 傳統解釋

災煞屬沖破之性，古人認為此煞主災厄纏身，尤以水火兩類災害為甚。凡命中見災煞者，應保持警覺，避免危險場所，居家注意防火防水，出行謹慎，可佩帶護身符化解。
