# 天醫

## 定義

天醫星依月支起算，取月支前一支（地支序向前退一位）為天醫所在，主醫療與健康之吉星。

## 類型

**吉星**

## 含義

主適合醫療行業、身體康健、逢病可癒。命中帶天醫星者，天生具備醫療天賦，對醫學、藥學、中醫等領域有特殊感悟，且本身健康運佳，即使生病也能較快康復。

## 起例方法

依**月支**查前一地支（月支退一位即為天醫）：

| 月支 | 天醫星 |
|------|-------|
| 子   | 亥     |
| 丑   | 子     |
| 寅   | 丑     |
| 卯   | 寅     |
| 辰   | 卯     |
| 巳   | 辰     |
| 午   | 巳     |
| 未   | 午     |
| 申   | 未     |
| 酉   | 申     |
| 戌   | 酉     |
| 亥   | 戌     |

> 規律：天醫星 = 月支的前一個地支（地支逆序退一位）。

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 祖上有醫緣，或幼年健康佳 |
| 月柱 | 事業與醫療相關，職業發展順遂 |
| 日柱 | 本人健康運強，適合從事醫療 |
| 時柱 | 晚年健康，子女有醫療緣 |

## 傳統解釋

天醫星象徵上天賦予的醫療能力與健康福分，古人認為命帶天醫者，前世曾行醫救人，今生得此星作為回報。此星入命，不論從事何種行業，遇病皆能得到良醫相助，轉危為安。若配合其他格局，更適合從事中西醫、藥師、護理、心理諮詢等與健康相關之職業。
