# 暴敗

## 定義

暴敗為八字神煞之一，屬**凶煞**。暴敗指財業突然崩潰，難以守成。命帶暴敗者，主突然敗落、錢財難守，宜謹慎保守，避免冒進。

## 含義

- 主突然敗落、錢財難守、宜謹慎
- 事業財富易在短時間內急速崩潰
- 興旺之後往往驟然落敗，難以守成

## 起例方式

依**日柱干支組合**判斷，以下特定干支組合即為暴敗日。

> **注意**：暴敗與**十惡大敗**為同一批日柱的不同稱謂，兩者干支組合完全相同。

### 暴敗日柱對照表

| 序號 | 日柱干支 | 五行組合說明 |
|------|---------|------------|
| 1  | 甲辰 | 木土，甲木克辰土，辰中藏水洩木 |
| 2  | 乙巳 | 木火，乙木生巳火，巳中庚金克乙 |
| 3  | 丙申 | 火金，丙火克申金，申中壬水克丙 |
| 4  | 丁亥 | 火水，丁火被亥水克，力量消耗 |
| 5  | 戊戌 | 土土，比旺卻內藏火金，耗散嚴重 |
| 6  | 己丑 | 土土，比旺卻丑中藏金水，財星洩 |
| 7  | 庚辰 | 金土，庚金得辰土生，然辰中藏水洩 |
| 8  | 辛巳 | 金火，辛金被巳火克，力量受損 |
| 9  | 壬申 | 水金，壬水得申金生，然申沖寅 |
| 10 | 癸亥 | 水水，比旺卻亥中藏甲木，洩耗財 |

### 範例

| 柱位 | 天干 | 地支 |
|------|------|------|
| 年柱 | 甲   | 子   |
| 月柱 | 丙   | 午   |
| 日柱 | **丙** | **申** |
| 時柱 | 壬   | 子   |

日柱為「丙申」，屬暴敗日（十惡大敗之一）。

## 各方面影響

| 影響層面 | 說明 |
|---------|------|
| 財運 | 積財不易，易有突發破財事件 |
| 事業 | 事業起伏劇烈，難以長久維持 |
| 健康 | 身體亦可能有突發狀況 |
| 應對之道 | 保守穩健，切忌投機冒險 |

## 傳統解說

暴敗之象，猶如繁盛之花驟然凋零，讓人措手不及。古書以「十惡大敗」稱此類日柱，認為其干支配合不佳，五行相剋之力強烈，難以守住財富與事業。命帶暴敗者，宜行事低調，量力而為，以守成為主，慎防突發之敗局。
