# 伏屍

## 定義

伏屍為八字神煞之一，屬**凶煞**。「伏屍」意指屍伏不起，與疾病、健康衰敗相關，主命主體弱多病、病患纏身。

## 含義

- 主疾病纏身、健康不佳
- 易有慢性病或難以痊癒之症
- 須特別注意養生保健
- 宜定期健康檢查，防患未然

## 起算方式

以**年支**查伏屍所在之地支，若該地支出現於命盤其他柱位，即為伏屍。

## 各柱位影響

| 柱位 | 影響說明 |
|------|----------|
| 年柱 | 幼年體質較弱，祖先或長輩多病 |
| 月柱 | 青壯年健康受損，父母宮有病患之象 |
| 日柱 | 本身健康較差，配偶亦須注意身體 |
| 時柱 | 晚年多病，子女健康需關注 |

## 傳統解說

伏屍古稱「屍伏不起」，意指人臥病在床、難以起身，象徵健康方面的障礙與阻滯。命帶伏屍者，一生之中容易與疾病為伴，體質較常人虛弱，宜注重日常保養、飲食調理，並避免過度操勞。若伏屍與病符、死符等煞同柱，則健康之憂更為明顯，應格外謹慎。
