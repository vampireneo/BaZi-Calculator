# 桃花

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 中性神煞 |
| 別名 | 咸池、桃花煞 |
| 取用基準 | 日支或年支所屬三合局 |

## 定義與含義

桃花為命盤中象徵人緣與異性緣的中性神煞。命帶桃花者，主人緣佳、異性緣好，外表具吸引力，待人親切，社交能力強。然須防桃色糾紛、感情糾葛，嚴重時可能影響婚姻與名譽。

## 起算方法

以**日支**或**年支**所屬的三合局為基準，查看四柱地支中是否出現對應地支。

| 日支或年支三合局 | 桃花地支 |
|----------------|---------|
| 申子辰（水局） | 酉 |
| 寅午戌（火局） | 卯 |
| 巳酉丑（金局） | 午 |
| 亥卯未（木局） | 子 |

## 判斷步驟

1. 找出日支（日柱地支）或年支（年柱地支）
2. 判斷其所屬三合局
3. 對照上表查出對應桃花地支
4. 在四柱地支中搜尋該地支，出現即為桃花

## 各柱位置效應

| 柱位 | 效應說明 |
|------|---------|
| 年柱 | 早年人緣佳，或家族中有桃花特質之人 |
| 月柱 | 職場人緣好，工作上異性緣旺，但易有是非 |
| 日柱 | 個人魅力強，感情豐富，配偶具桃花特質 |
| 時柱 | 晚年仍有異性緣，子女人緣佳 |

## 桃花類型

| 類型 | 說明 |
|------|------|
| 內桃花 | 桃花在日支，感情較專注 |
| 外桃花 | 桃花在年、月、時支，容易外遇或感情複雜 |
| 牆內桃花 | 桃花坐祿或旺地，多為正當感情 |
| 牆外桃花 | 桃花坐敗地，多為不正當感情 |

## 傳統解說

桃花又稱「咸池」，古籍以咸池為水之聚集之地，主慾望與情感。桃花入命並非全然不吉，若桃花有吉星相輔（如天乙貴人、文昌），則人緣出眾、事業得助；若桃花逢沖逢合，或與羊刃、劫財同柱，則感情易生波折，需格外謹慎。

> **注意**：桃花為中性神煞，吉凶需視整體命盤格局而定，不可單一論斷。
