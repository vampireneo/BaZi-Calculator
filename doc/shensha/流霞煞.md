# 流霞煞

## 定義

流霞煞依日干查對應地支，為主血光之災的凶煞，女命尤需重視。

## 類型

**凶煞**

## 含義

主血光之災、女命須防難產、男命防意外。流霞煞如霞光流散，象徵血液流失之象，命中帶此煞者，須防手術、意外出血等血光之事，女性尤應注意生產安全。

## 起例方法

依**日干**查對應地支，若該地支出現於四柱中即帶流霞煞：

| 日干 | 流霞煞地支 |
|------|-----------|
| 甲   | 酉         |
| 乙   | 戌         |
| 丙   | 未         |
| 丁   | 申         |
| 戊   | 未         |
| 己   | 申         |
| 庚   | 巳         |
| 辛   | 午         |
| 壬   | 卯         |
| 癸   | 辰         |

## 各柱影響

| 所在位置 | 影響說明 |
|----------|----------|
| 年柱 | 幼年曾有血光，或家族有手術史 |
| 月柱 | 中年須防外傷或手術 |
| 日柱 | 本人血光之煞最重，宜提前化解 |
| 時柱 | 晚年注意健康，防外科手術 |

## 傳統解釋

流霞煞在古籍中被視為主血光、手術的重要凶煞，尤其對女性影響深遠。古人認為女命日干逢流霞，生產時風險較高，須提前做好準備。男命帶此煞，則易遭刀傷或意外出血。現代命理上，此煞與手術、獻血等事件相關，主動進行的手術（如整形）有時可化解被動血光。
