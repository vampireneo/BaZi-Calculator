# 孤鸞煞

## 定義

孤鸞煞（Gū Luán Shā）為八字神煞之一，屬**凶煞**。此煞主孤獨，對婚姻感情影響甚大，命帶此煞者婚姻易不順，夫妻易分離，宜晚婚。

## 含義

- 主婚姻不順、夫妻易分離
- 感情生活波折較多，難以長久
- 宜晚婚，早婚易生變故
- 性格易孤傲，難與伴侶長期磨合

## 起法（以日柱干支查）

孤鸞煞以**特定日柱干支組合**論之，以下日柱即帶此煞：

| 日柱組合 | 五行說明 |
|---------|---------|
| 乙巳 | 木坐火，陰干 |
| 丁巳 | 火坐火，陰干 |
| 辛亥 | 金坐水，陰干 |
| 戊申 | 土坐金，陽干 |
| 壬寅 | 水坐木，陽干 |
| 戊午 | 土坐火，陽干 |
| 壬子 | 水坐水，陽干 |
| 丙午 | 火坐火，陽干 |
| 丙子 | 火坐水，陽干 |

**查法**：對照自身日柱干支組合，若符合上表任一組合，即命帶孤鸞煞。

## 各柱影響

| 位置 | 影響 |
|------|------|
| 日柱帶煞 | 婚姻感情最受影響，易孤獨或分離 |
| 命局有合化 | 煞力可減輕，婚姻較為穩固 |
| 命局無解 | 孤獨之象明顯，感情波折較多 |

## 傳統解說

孤鸞煞主孤獨之象，古籍云：「孤鸞入命，婚姻多磨，夫妻難偕老。」命帶此煞者，感情路上多有波折，伴侶間易生嫌隙或分離。然煞星之力可藉命局其他吉神化解，若命局有貴人星、天乙等吉星調和，孤鸞之凶力可大為減輕。宜晚婚，待命局運程成熟後再論婚嫁為宜。
